ཇི་ལྟར་སོ་སོར་དྲིས་ནས་དགེ་སློང་གིས་ལན་བཏབ་པ་དེ་བཞིན་དུ་དགེ་སློང་གི་འཁོར་འདི་ལྟ་བུ་ཡང་ལན་གསུམ་གྱི་བར་དུ་བསྒྲག་པར་བྱ་བ་ཡིན་ནོ། །ཡང་དགེ་སློང་གང་དགེ་སློང་གི་འཁོར་འདི་ལྟ་བུར་ལན་གསུམ་གྱི་བར་དུ་བསྒྲགས་པ་ན་ལྟུང་བ་ཡོད་ ལ་དྲན་བཞིན་དུ་མི་མཐོལ་ན་དེ་ཤེས་བཞིན་དུ་བརྫུན་དུ་སྨྲ་བ་ཡིན་ནོ། །ཚེ་དང་ལྡན་པ་དག་ཤེས་བཞིན་དུ་བརྫུན་དུ་སྨྲ་བ་ནི་བཅོམ་ལྡན་འདས་ཀྱིས་བར་དུ་གཅོད་པའི་ཆོས་སུ་གསུངས་སོ། །དེ་བས་ན་དགེ་སློང་གང་ལྟུང་བ་བྱུང་བ་རྣམ་པར་དག་པར་འདོད་པས་ལྟུང་བ་ཡོད་ལ་དྲན་ཞིང་མཐོང་བ་མཐོལ་བར་བྱའོ། །མཐོལ་ན་དེ་བདེ་བ་ལ་རེག་བར་གནས་པར་འགྱུར་རོ། །མ་མཐོལ་མ་བཤགས་ན་ནི་མི་འགྱུར་རོ། །
# f2.3  གླེང་གཞི་འདོན་པའི་མཇུག་བསྡུ་བ་ནི།

# f2.4  རྣམ་པར་བཞག་པ་བསྒྲགས་པ་ནི།

# f3.a སྒྲོ་སྐུར་སྤང་བའི་དོན་དུ་སྡོམ་ནི།
སྡོམ་ལ། མི་ཚངས་སྤྱོད་དང་རྐུ་བ་དང། །མི་ལ་གསད་པར་མི་བྱ་བ། །བརྫུན་དུ་སྨྲ་དང་བཅས་པ་ཡི། །ཆོས་བཞི་འདིར་ནི་གསུངས་པ་ཡིན། །
# f3.b ལུང་ལ་གུས་པ་བསྐྱེད་པའི་དོན་དུ་ལུང་འབོག་པའི་དོན་ནི།
ཚེ་དང་ལྡན་པ་དག་ཕམ་པར་འགྱུར་བའི་ཆོས་བཞི་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་གདོན་པ་ལས་འབྱུང་ངོ། །
# f3.c.1 མི་ཚངས་པར་སྤྱོད་པའི་ཕམ་པ།
ཡང་དགེ་སློང་གང་དགེ་སློང་རྣམས་དང་ལྷན་ཅིག་བསླབ་པ་མཚུངས་པར་གྱུར་པས་བསླབ་པ་མ་ཕུལ་བསླབ་པ་ཉམས་པར་མ་བྱས་པར་མི་ཚངས་པར་སྤྱོད་པ་འཁྲིག་པའི་ཆོས་བསྟེན་ན། ཐ་ན་དུད་འགྲོའི་སྐྱེ་གནས་སུ་སྐྱེས་པ་དང་ལྷན་ཅིག་ཀྱང་རུང་སྟེ། དགེ་སློང་དེ་ཕམ་པར་གྱུར་པ་ཡིན་གྱིས་གནས་པར་མི་བྱའོ། །
# f3.c.2 གཉིས་པ་མ་བྱིན་ལེན་ནི།
ཡང་དགེ་སློང་གང་གཞན་དག་གི་གྲོང་ན་འདུག་པ་འམ། དགོན་པ་ན་འདུག་པ་མ་བྱིན་པར་རྐུ་བའི་གྲངས་སུ་གཏོགས་པ་བླངས་ན། ཇི་ཙམ་མ་བྱིན་པར་བླངས་པ་དེ་རྒྱལ་པོ་འམ། བློན་པོ་ཆེན་པོས་བཟུང་ནས་དེ་ལ་འདི་སྐད་ཅེས། ཀྱེ་མི་ཁྱོད་ནི་རྐུན་མ་འོ། །བྱིས་པའོ། །བླུན་པའོ། །རྐུ་བའོ་ཞེས་ཟེར་ཞིང་གསོད་དམ། འཆིང་ངམ། སྤྱུགས་ཀྱང་རུང་སྟེ། དགེ་སློང་དེ་ལྟར་མ་བྱིན་པར་ལེན་ན་དགེ་སློང་དེ་ཡང་ཕམ་པར་གྱུར་པ་ཡིན་གྱིས་གནས་པར་མི་བྱའོ། །
# f3.c.3 གསུམ་པ་མི་ལ་བསད་པར་མི་བྱ་བ་ནི།
ཡང་དགེ་སློང་གང་མི་འམ་མིར་ཆགས་པ་ལ་བསམས་བཞིན་དུ་རང་གི་ལག་དར་ཏེ་སྲོག་བཅད་དམ། དེ་ལ་མཚོན་བྱིན་ནམ། དེ་ལ་མཚོན་ཐོགས་པ་གཉེར་ཏམ། དེ་འཆིར་བཅུག་གམ། དེ་ལ་འཆི་བའི་བསྔགས་པ་བརྗོད་ཀྱང་རུང་སྟེ། དེ་ལ་འདི་སྐད་ཅེས་ཀྱེ་མི་ཁྱོད་འཚོ་བ་སྡིག་པ་མི་གཙང་བ་ངན་པ་འདིས་ཅི་ཞིག་བྱ། ཀྱེ་མི་ཁྱོད་གསོན་པ་བས་ཤི་བླའོ་ཟེར་ཞིང། སེམས་ཀྱི་འདོད་པ་དང། སེམས་ཀྱི་ཀུན་ཏུ་རྟོག་པ་དག་གིས་རྣམས་གྲངས་དུ་མས་དེ་འཆིར་བཅུག་གམ། དེ་ལ་འཆི་བའི་བསྔགས་པ་བརྗོད་དེ། དེ་ཡང་རྩོམ་པ་དེས་དུས་བྱས་ན། དགེ་སློང་དེ་ཡང་ཕམ་པར་གྱུར་པ་ཡིན་གྱིས་གནས་པར་མི་བྱའོ། །
# f3.c.4 བཞི་པ་བརྫུན་དུ་སྨྲ་བའི་ཕམ་པ་ལ།
ཡང་དགེ་སློང་གང་མངོན་པར་མི་ཤེས་ཤིང་ཡོངས་སུ་མི་ཤེས་ལ་མིའི་ཆོས་བླ་མ་མཐའ་དང་། འཕགས་པ་དང། བྱེ་བྲག་ཐོབ་པ་དང། ཤེས་པ་དང། མཐོང་བ་དང། རེག་པར་སྤྱོད་པ་མེད་ལ་མེད་བཞིན་དུ་འདི་ཤེས་སོ། །འདི་མཐོང་ངོ་ཞེས་ཁས་འཆེས་པ་ལས། དེ་ལྟུང་བ་བྱུང་བ་རྣམ་པར་དག་པར་འདོད་ནས་དུས་གཞན་ཞིག་ན་དྲིས་ཀྱང་རུང། མ་དྲིས་ཀྱང་རུང་འདི་སྐད་ཅེས་ཚེ་དང་ལྡན་པ་དག་བདག་གིས་ནི་མི་ཤེས་པར་ཤེས་སོ་ཞེས་སྨྲས། མ་མཐོང་བར་མཐོང་ངོ་ཞེས་སྨྲས་ཏེ། གསོབ་གསོག་བརྫུན་དུ་སྨྲས་སོ་ཞེས་ཟེར་ན། མངོན་པའི་ང་རྒྱལ་མ་གཏོགས་ཏེ། དགེ་སློང་དེ་ཡང་ཕམ་པར་གྱུར་པ་ཡིན་གྱིས་གནས་པར་མི་བྱའོ། །
# f3.d ཕམ་པ་འདོན་པའི་མཇུག་བསྡུ་བ་ནི།
ཚེ་དང་ལྡན་པ་དག་བདག་གིས་ཕམ་པར་གྱུར་པའི་ཆོས་བཞི་པོ་དག་བཏོན་ཟིན་ཏོ། །
# f3.e ཕམ་པ་སྤང་བའི་ཆེད་དུ་ཉེས་དམིགས་བསྟན་པ་ནི།
དགེ་སློང་གིས་དེ་དག་ལས་ལྟུང་བ་གང་ཡང་རུང་བ་ཞིག་བྱས་ན་ཐོག་མ་ཇི་ལྟ་བར་ཕྱིས་ཀྱང་དེ་བཞིན་ཕམ་པར་གྱུར་པ་ཡིན་ཏེ། དགེ་སློང་རྣམས་དང་ལྷན་ཅིག་གནས་པ་དང་ལོངས་སྤྱོད་དུ་མི་དབང་གིས་གནས་པར་མི་བྱའོ། །
# f3.f དག་པ་ཉིད་དུ་རྟོགས་པར་བྱ་བའི་རྣམ་བཞག་བསྒྲག་པ་ནི།
དེ་ལ་བདག་གིས་ཚེ་དང་ལྡན་པ་དག་ལ་ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ། ཞེས་ལན་གཉིས་ལན་གསུམ་དུ་དྲིའོ། །འདི་ལ་ཚེ་དང་ལྡན་པ་དག་ཡོངས་སུ་དག་ན་འདི་ལྟར་ཅང་མི་སྨྲ་བས་དེ་དེ་བཞིན་དུ་འཛིན་ཏོ། །

# f4.a གཉིས་པ་ལྷག་མ་འདོན་པ་ལ། སྒྲོ་སྐུར་སྤང་བའི་དོན་དུ་སྡོམ་ནི།
སྡོམ་ལ། ཁུ་བ་འཛིན་པ་འཁྲིག་ཚིག་བསྙེན་བཀུར་སྨྱན། །ཁང་པ་ཁང་ཆེན་དང་ནི་གཞི་མེད་པ། །བག་ཙམ་དགེ་འདུན་དབྱེན་དང་དེ་རྗེས་ཕྱོགས། །ཁྱིམ་སུན་འབྱིན་དང་བཀའ་བློ་མི་བདེ་བའོ། །
# f4.b ལུང་ལ་གུས་པ་བསྐྱེད་པའི་དོན་དུ་འདོན་པ་ལུང་འབོག་པའི་དོན་ནི།
ཚེ་དང་ལྡན་པ་དག་དགེ་འདུན་ལྷག་མའི་ཆོས་བཅུ་གསུམ་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# f4.c.1 ལྷག་མ་དང་པོའི་སྐབས་བབ་ཀྱི་གཞི་བསམ་པ་དང་བཅས་པ་ནི།
བསམས་བཞིན་དུ་ཁུ་བ་ཕྱུང་ན་རྨི་ལམ་གྱི་མ་གཏོགས་ཏེ་དགེ་འདུན་ལྷག་མ་འོ། །
# f4.c.2 གཉིས་པ་འཛིན་པ་ལ།
ཡང་དགེ་སློང་གང་དྲལ་ཅིང་གྱུར་པའི་སེམས་ཀྱིས་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་ལུས་རེག་པར་བྱེད་དམ། ལག་པ་ནས་བཟུང་ངམ། དཔུང་པ་ནས་བཟུང་ངམ། ལན་བུ་ནས་བཟུང་ངམ། ཡན་ལག་དང་ཉིང་ལག་གང་ཡང་རུང་བ་ལ་ནོམ་པ་དང་ཉུག་པ་བདག་གིར་བྱེད་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.3 གསུམ་པ་འཁྲིག་ཚིག་ལ།
ཡང་དགེ་སློང་གང་དྲལ་ཅིང་གྱུར་པའི་སེམས་ཀྱིས་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་གནས་ངན་ལེན་གྱི་ཚིག་སྡིག་པ་ཅན་ཚོགས་པར་མི་དབྱུང་བ་འཁྲིག་པ་ལས་བྱུང་བ་དང་ལྡན་པ་དག་སྐྱེས་བུས་ན་ཆུང་ལ་ཇི་ལྟ་བ་བཞིན་དུ་སྨྲས་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.4 བཞི་པ་བསྙེན་བཀུར་ནི།
ཡང་དགེ་སློང་གང་དྲལ་ཅིང་གྱུར་པའི་སེམས་ཀྱིས་བུད་མེད་ཀྱི་ལུས་ཀྱི་འདུན་དུ་བདག་ཉིད་ཀྱི་ལུས་ཀྱི་བསྙེན་བཀུར་བྱ་བའི་ཕྱིར་འདི་ལྟ་སྟེ། ང་ལྟ་བུའི་དགེ་སློང་ཚུལ་ཁྲིམས་དང་ལྡན་པ། དགེ་བའི་ཆོས་ཅན། ཚངས་པར་སྤྱོད་པ་ལ་འདི་ལྟར་འཁྲིག་པ་ལས་བྱུང་བ་དང་ལྡན་པའི་ཆོས་འདིས་བསྙེན་བཀུར་བྱས་ན། སྲུ་འདི་ནི་བསྙེན་བཀུར་བྱས་པ་རྣམས་ཀྱི་ནང་ན་མཆོག་ཡིན་ནོ་ཞེས་བསྔགས་པ་བརྗོད་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.5 ལྔ་པ་གཉེན་ལ་གདུང་བ་ནི།
ཡང་དགེ་སློང་གང་བུད་མེད་ལ་སྐྱེས་པའི་ཚིག་དང། སྐྱེས་པ་ལ་བུད་མེད་ཀྱི་ཚིག་གིས་ཆུང་མ་ཉིད་དང། མཛའ་ན་མོ་ཉིད་དུ་སྨྱན་བྱེད་ན་ཐ་ན་ཐང་འགའ་ཕྲད་པ་ཡང་རུང་སྟེ་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.6 དྲུག་པ་ཁང་པ་ལ།
དགེ་སློང་གིས་བདག་གིས་བསླངས་པ། བདག་པོ་མེད་པ། བདག་གི་ཕྱིར་ཁང་པ་རྩིག་ཏུ་འཇུག་ན་དགེ་སློང་དེས་ཁང་པ་ཚད་བཞིན་དུ་བརྩིག་ཏུ་ཆུག་ཅིག །དེ་ལ་ཁང་པའི་ཚད་ནི་འདི་ཡིན་ཏེ། ནང་གི་སྲིད་དུ་བདེ་བར་གཤེགས་པའི་མཐོའི་མཐོ་བཅུ་གཉིས། ཞེང་དུ་མཐོ་བདུན་ནོ། །གཞི་བལྟ་བའི་ཕྱིར་དགེ་སློང་དེས་དགེ་སློང་དག་བཀྲི་བར་བྱའོ། །ཁྲིད་པའི་དགེ་སློང་དག་གིས་ཀྱང་གཞི་རུང་བ་དང། རྩོད་པ་མེད་པ་དང། བརྩམ་དུ་རུང་བར་བལྟ་བར་བྱའོ། །
གལ་ཏེ་དགེ་སློང་གིས་གཞི་མི་རུང་བ་འམ། རྩོད་པ་དང་བཅས་པ་འམ། བརྩམ་དུ་མི་རུང་བར་བདག་གིས་བསླངས་པ་བདག་པོ་མེད་པ་བདག་གི་ཕྱིར་ཁང་པ་རྩིག་ཏུ་འཇུག་གམ། གཞི་བལྟ་བའི་ཕྱིར་དགེ་སློང་དག་ཀྱང་མི་ཁྲིད་དམ། དགེ་སློང་ཁྲིད་པ་དག་ལ་གཞི་སྟོན་ཏམ། ཚད་ལས་འདས་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.7 བདུན་པ་ཁང་ཆེན་ནི།
དགེ་སློང་གིས་བདག་པོ་ཡོད་པ་དགེ་འདུན་གྱི་ཕྱིར་གཙུག་ལག་ཁང་ཆེན་པོ་རྩིག་ཏུ་འཇུག་ན་གཞི་བལྟ་བའི་ཕྱིར་དགེ་སློང་དེས་དགེ་སློང་དག་བཀྲི་བར་བྱའོ། །ཁྲིད་པའི་དགེ་སློང་དག་གིས་ཀྱང་གཞི་རུང་བ་དང། རྩོད་པ་མེད་པ་དང། བརྩམ་དུ་རུང་བར་བལྟ་བར་བྱའོ། །གལ་ཏེ་དགེ་སློང་གིས་གཞི་མི་རུང་བ་འམ། རྩོད་པ་དང་བཅས་པ་འམ། བརྩམ་དུ་མི་རུང་བར་བདག་པོ་ཡོད་པ་དགེ་འདུན་གྱི་ཕྱིར་གཙུག་ལག་ཁང་ཆེན་པོ་རྩིག་ཏུ་འཇུག་གམ། གཞི་བལྟ་བའི་ཕྱིར་དགེ་སློང་དག་ཀྱང་མི་ཁྲིད་དམ། དགེ་སློང་ཁྲིད་པ་དག་ལ་གཞི་མི་སྟོན་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.8 བརྒྱད་པ་གཞི་མེད་ལ།
ཡང་དགེ་སློང་གང་ཁྲོས་ཤིང་ཞེ་སྡང་བར་གྱུར་ནས། ཅི་ནས་ཀྱང་འདི་ཚངས་པར་སྤྱོད་པ་དང་དབྲལ་ལོ་སྙམ་ནས། དགེ་སློང་དག་པ་ལྟུང་བ་མེད་པ་ལ། གཞི་མེད་པར་ཕམ་པར་འགྱུར་བའི་ཆོས་ ཀྱིས་སྐུར་པ་ལས་དེ་དུས་གཞན་ཞིག་ན་དྲིས་ཀྱང་རུང། མ་དྲིས་ཀྱང་རུང། རྩོད་པ་དེ་ཡང་གཞི་མེད་པ་ཡིན་ལ། དགེ་སློང་ཡང་ཞེ་སྡང་ལ་གནས་པས་ཞེ་སྡང་གིས་སྨྲས་སོ་ཞེ་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.9 དགུ་པ་བག་ཙམ་ལ།
ཡང་དགེ་སློང་གང་ཁྲོས་ཤིང་ཞེ་སྡང་བར་གྱུར་ནས་ཅི་ནས་ཀྱང་འདི་ཚངས་པར་སྤྱོད་པ་དང་དབྲལ་ལོ་སྙམ་ནས། དགེ་སློང་དག་པ་ལྟུང་བ་མེད་པ་ལ་གཞན་གྱི་ཆ་མ་ཡིན་པ་དང་འཐུན་པ་ཕམ་པར་འགྱུར་བའི་ཆོས་ཀྱིས་སྐུར་པ་ལས། དེ་དུས་གཞན་ཞིག་ན། དྲིས་ཀྱང་རུང། མ་དྲིས་ཀྱང་རུང། རྩོད་པ་དེ་ཡང་གཞན་གྱི་ཆ་མ་ཡིན་པ་དང་འཐུན་པ་ཡིན་ལ། རྩོད་པ་དེ་གཞན་གྱི་ཆ་མ་ཡིན་པ་དང་འཐུན་པས་བག་ཙམ་ལས་བསམས་པ་ཙམ་གྱིས་ཆོས་འགའ་ཞིག་བླངས་པར་གྱུར་ལ། དགེ་སློང་ཡང་ཞེ་སྡང་ལ་གནས་པས་ཞེ་སྡང་གིས་སྨྲས་སོ་ཞེ་ན་དགེ་འདུན་ལྷག་མ་འོ། །
# f4.c.10 བཅུ་པ་དགེ་འདུན་དབྱེན་ལ།
ཡང་དགེ་སློང་གང་དགེ་འདུན་འཐུན་པ་དབྱེ་བའི་ཕྱིར་རྟུལ་བར་བྱེད་ཅིང། དབྱེན་བྱེད་པར་གྱུར་པའི་རྩོད་པ་ཡང་དག་པར་བླངས་ནས་རབ་ཏུ་བཟུང་སྟེ་འདུག་ན། དགེ་སློང་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཁྱོད་དགེ་འདུན་འཐུན་པ་དབྱེ་བའི་ཕྱིར་རྟུལ་བར་མ་བྱེད་ཅིག །དབྱེན་བྱེད་པར་འགྱུར་བའི་རྩོད་པ་ཡང་དག་པར་བླངས་ནས་རབ་ཏུ་བཟུང་སྟེ་མ་འདུག་ཅིག །ཚེ་དང་ལྡན་པ་དགེ་འདུན་དང་འཐུན་པར་གྱིས་ཤིག །དགེ་འདུན་འཐུན་མི་ཕྱེད། ཀུན་དུ་དགའ་མི་རྩོད། མཆོག་གཅིག་འདོན་པ་གཅིག་ཅིང་ཆུ་དང་འོ་མ་འདྲེས་པ་ལྟ་བུར་གྱུར་ལ། སྟོན་པའི་བསྟན་པ་ཡང་གསལ་བར་བྱེད་ན་བདེ་བ་ལ་རེག་པར་འགྱུར་གྱིས། ཚེ་དང་ལྡན་པ་ཁྱོད་དགེ་འདུན་འབྱེད་པར་བྱེད་པའི་གཞི་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་སློང་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་བསྒོ་བ་ན། གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱའོ། །ཡང་དག་པར་བསྟན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་དགེ་འདུན་ལྷག་མ་འོ། །
# f4.c.11 བཅུ་གཅིག་པ་དེའི་རྗེས་སུ་ཕྱོགས་པ་ནི།
དགེ་སློང་དེའི་གྲོགས་བྱེད་པའི་དགེ་སློང་མི་མཐུན་པར་སྨྲ་བ་ཉིད་ཀྱི་རྗེས་སུ་ཕྱོགས་པ་གཅིག་གམ་གཉིས་སམ་མང་པོ་དག་ཡོད་ཅིང། གལ་ཏེ་དེ་དག་དགེ་སློང་རྣམས་ལ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་ཅག་དགེ་ཡང་རུང་། སྡིག་ཀྱང་རུང། དགེ་སློང་འདི་ལ་ཅི་ཡང་མ་སྨྲ་ཤིག །དེ་ཅིའི་ཕྱིར་ཞེ་ན། ཚེ་དང་ལྡན་པ་དག་དགེ་སློང་འདི་ནི་ཆོས་སྨྲ་བ། འདུལ་བ་སྨྲ་བ། དགེ་སློང་འདི་ནི་ཆོས་དང་འདུལ་བ་ཡང་དག་པར་བླངས་ནས་རབ་ཏུ་བཟུང་སྟེ་རྗེས་སུ་ཐ་སྙད་འདོགས་པར་བྱེད་པ། དགེ་སློང་འདི་ནི་ཤེས་བཞིན་དུ་སྨྲའི་མི་ཤེས་པར་མ་ཡིན་པའི་ཕྱིར་ཏེ། དགེ་སློང་འདི་གང་ལ་འདོད་ཅིང་བཟོད་པ་དེ་ལ་བདག་ཅག་ཀྱང་འདོད་ཅིང་བཟོད་དོ་ཞེས་ཟེར་ན། དགེ་སློང་དེ་དག་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་ཅག་དེ་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་ཅག་དགེ་ཡང་རུང། སྡིག་ཀྱང་རུང་། དགེ་སློང་འདི་ལ་ཅི་ཡང་མ་སྨྲ་ཤིག །དེ་ཅིའི་ཕྱིར་ཞེ་ན། ཚེ་དང་ལྡན་པ་དག་དགེ་སློང་འདི་ནི་ཆོས་སྨྲ་བ་འདུལ་བ་སྨྲ་བ། དགེ་སློང་འདི་ནི་ཆོས་དང་འདུལ་བ་ཡང་དག་པར་བླངས་ནས་རབ་ཏུ་བཟུང་སྟེ་རྗེས་སུ་ཐ་སྙད་འདོགས་པར་བྱེད་ལ། དགེ་སློང་འདི་ནི་ཤེས་བཞིན་དུ་སྨྲའི་མི་ཤེས་པར་མ་ཡིན་པའི་ཕྱིར་ཏེ། དགེ་སློང་འདི་གང་ལ་འདོད་ཅིང་བཟོད་པ་དེ་ལ་བདག་ཅག་ཀྱང་འདོད་ཅིང་བཟོད་དོ་ཞེས་མ་ཟེར་ཅིག །དེ་ཅིའི་ཕྱིར་ཞེ་ན། ཚེ་དང་ལྡན་པ་དག་དགེ་སློང་འདི་ནི་ཆོས་སྨྲ་བ་མ་ཡིན། འདུལ་བ་སྨྲ་བ་མ་ཡིན་གྱི། ཚེ་དང་ལྡན་པ་དགེ་སློང་འདི་ནི་ཆོས་མ་ཡིན་པ་སྨྲ་བ། འདུལ་བ་མ་ཡིན་པ་སྨྲ་བ། དགེ་སློང་འདི་ནི་ཆོས་མ་ཡིན་པ་དང་འདུལ་བ་མ་ཡིན་པ་ཡང་དག་པར་བླངས་ནས་རབ་ཏུ་བཟུང་སྟེ་རྗེས་སུ་ཐ་སྙད་འདོགས་པར་བྱེད་པ། དགེ་སློང་འདི་ནི་མི་ཤེས་བཞིན་དུ་སྨྲའི་ཤེས་པར་སྨྲ་བ་མ་ཡིན་པའི་ཕྱིར་ཏེ། དགེ་སློང་འདི་ནི་གང་ལ་འདོད་ཅིང་བཟོད་པ་དེ་ལ་ཚེ་དང་ལྡན་པ་དག་འདོད་ཅིང་བཟོད་པར་མ་བྱེད་ཅིག །ཚེ་དང་ལྡན་པ་དག་ཁྱེད་དགེ་འདུན་འབྱེད་འདོད་པར་མ་བྱེད་པར་ཚེ་དང་ལྡན་པ་དག་དགེ་འདུན་འཐུན་པ་ཉིད་དུ་འདོད་པར་གྱིས་ཤིག །ཚེ་དང་ལྡན་པ་དག་དགེ་འདུན་འཐུན་པར་གྱིས་ཤིག །དགེ་འདུན་འཐུན་མི་ཕྱེད། ཀུན་ཏུ་དགའ་མི་རྩོད། མཆོག་གཅིག་འདོན་པ་གཅིག་ཅིང་ཆུ་དང་འོ་མ་གཅིག་ཏུ་འདྲེས་པ་ལྟ་བུར་གྱུར་ལ། སྟོན་པའི་བསྟན་པ་གསལ་བར་བྱེད་ན་བདེ་བ་ལ་རེག་པར་གནས་པར་འགྱུར་གྱིས། ཚེ་དང་ལྡན་པ་དགེ་འདུན་འབྱེད་པའི་རྗེས་སུ་ཕྱོགས་ཤིང་མི་མཐུན་པར་སྨྲ་བ་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་སློང་དེ་དག་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་ཅེས་བསྒོ་བ་ན། གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱ། ཡང་དག་པར་བསྟན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.12 བཅུ་གཉིས་པ་ཁྱིམ་སུན་འབྱིན་པ་ལ།
དགེ་སློང་རབ་ཏུ་མང་པོ་དག་གྲོང་ངམ་གྲོང་རྡལ་ཞིག་ན་ཉེ་བར་རྟེན་ཅིང་གནས་པར་གྱུར་ལ། དེ་དག་ཀྱང་ཁྱིམ་སུན་འབྱིན་པ་སྡིག་པའི་ཆོས་ཀུན་ཏུ་སྤྱོད་པར་གྱུར་ཅིང། དེ་དག་གིས་ཁྱིམ་དག་སུན་ཕྱུང་བར་མཐོང་ངམ། ཐོས་སམ། རབ་ཏུ་ཤེས་དེ་དག་སྡིག་པ་ཀུན་དུ་སྤྱོད་པར་མཐོང་ངམ། ཐོས་སམ། རབ་ཏུ་ཤེས་ན་དགེ་སློང་དེ་དག་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་ཁྱིམ་སུན་འབྱིན་པ་སྡིག་པའི་ཆོས་ཀུན་དུ་སྤྱོད་པ་དག་ཡིན་ཏེ། ཁྱེད་ཀྱིས་ཁྱིམ་དག་སུན་ཕྱུང་བར་ཡང་མཐོང་ཞིང་ཐོས་ལ་རབ་ཏུ་ཤེས། ཁྱོད་སྡིག་པ་ཀུན་ཏུ་སྤྱོད་པར་ཡང་མཐོང་ཞིང་ཐོས་ལ་རབ་ཏུ་ཤེས་ཀྱིས། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་འདི་ན་གནས་པས་ཆོག་གིས་གནས་འདི་ནས་དེང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །གལ་ཏེ་དེ་དག་དགེ་སློང་རྣམས་ལ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་འདི་ན་དགེ་སློང་ཁ་ཅིག་འདུན་པས་འགྲོ་བ། ཞེ་སྡང་གིས་འགྲོ་བ། གཏི་མུག་གིས་འགྲོ་བ། འཇིགས་པས་འགྲོ་བ་དག་ཡིན་ཏེ། འདི་ལྟར་ལྟུང་བ་འདྲ་བ་ཁོ་ན་ལས་དགེ་སློང་ཁ་ཅིག་ནི་སྐྲོད་པར་བྱེད། ཁ་ཅིག་ནི་སྐྲོད་པར་མི་བྱེད་དོ། །ཞེས་ཟེར་ན་དགེ་སློང་དེ་དག་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་དེ་སྐད་ཅེས་ཚེ་དང་ལྡན་པ་དག་འདི་ན་དགེ་སློང་ཁ་ཅིག་འདུན་པས་འགྲོ་བ། ཞེ་སྡང་གིས་འགྲོ་བ། གཏི་མུག་གིས་འགྲོ་བ། འཇིགས་པས་འགྲོ་བ་དག་ཡིན་ཏེ། འདི་ལྟར་ལྟུང་བ་འདྲ་བ་ཁོ་ན་ལས་དགེ་སློང་ཁ་ཅིག་ནི་སྐྲོད་པར་བྱེད། ཁ་ཅིག་ནི་སྐྲོད་པར་མི་བྱེད་དོ་ཞེས་མ་ཟེར་ཅིག །དེ་ཅིའི་ཕྱིར་ཞེ་ན། དགེ་སློང་འདི་དག་ནི་འདུན་པས་འགྲོ་བ་མ་ཡིན། ཞེ་སྡང་གིས་འགྲོ་བ་མ་ཡིན། གཏི་མུག་གིས་འགྲོ་བ་མ་ཡིན། འཇིགས་པས་འགྲོ་བ་དག་མ་ཡིན་གྱི། འདི་ལྟར་ཚེ་དང་ལྡན་པ་དག་ཉིད་ཁྱིམ་སུན་འབྱིན་པ་སྡིག་པའི་ཆོས་ཀུན་ཏུ་སྤྱོད་པ་དག་ཡིན་ཏེ། ཁྱེད་ཉིད་ཀྱིས་ཁྱིམ་དག་སུན་ཕྱུང་བར་ཡང་མཐོང་ཞིང་ཐོས་ལ་རབ་ཏུ་ཤེས། ཁྱེད་སྡིག་པ་ཀུན་དུ་སྤྱོད་པར་ཡང་མཐོང་ཞིང་ཐོས་ལ་རབ་ཏུ་ཤེས་པའི་ཕྱིར་ཏེ། ཚེ་དང་ལྡན་པ་དག་འདུན་པས་འགྲོ་བ་ཞེས་བྱ་བའི་ཚིག་གི་ལམ་དང། ཞེ་སྡང་གིས་འགྲོ་བ། གཏི་མུག་གིས་འགྲོ་བ། འཇིགས་པས་འགྲོ་བ་ཞེས་བྱ་བའི་ཚིག་གི་ལམ་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་སློང་དེ་དག་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་ཅེས་བསྒོ་བ་ན་གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱ། ཡང་དག་པར་བསྟན་པར་བྱ་འོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.c.13 བཅུ་གསུམ་པ་བཀའ་བློ་མི་བདེ་བ་ལ།
འདི་ན་དགེ་སློང་འགའ་ཞིག་བཀའ་བློ་མི་བདེ་བའི་རང་བཞིན་ཅན་དུ་གྱུར་ལ། དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་གདོན་པར་གཏོགས་པ་བསླབ་པའི་གཞི་རྣམས་དང། བདེ་བར་གཤེགས་པའི་མདོར་གཏོགས་པ་རྣམས་ཀྱིས་ཆོས་དང་མཐུན་པ་དང། འདུལ་བ་དང་མཐུན་པར་སྨྲས་པ་ན། ཚེ་དང་ལྡན་པ་དག་ཁྱེད་ཅག་དགེ་ཡང་རུང་། སྡིག་ཀྱང་རུང། བདག་ལ་ཅི་ཡང་མ་སྨྲ་ཤིག །བདག་ཀྱང་དགེ་ཡང་རུང་། སྡིག་ཀྱང་རུང། ཚེ་དང་ལྡན་པ་དག་ལ་ཅི་འང་མི་སྨྲའོ། །ཚེ་དང་ལྡན་པ་དག་གིས་བདག་ལ་ཚིག་གི་ལམ་ཐོང་ཤིག །ཁྱེད་ལ་ཡང་བདག་གིས་སྨྲས་པས་ཅི་ཞིག་བྱ་ཞེས་བདག་ཉིད་བརྗོད་པར་བྱ་བ་མ་ཡིན་པར་བྱེད་ན། དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཁྱོད་དགེ་སློང་རྣམས་ཀྱིས་གདོན་པར་གཏོགས་པ་བསླབ་པའི་གཞི་རྣམས་དང་། བདེ་བར་གཤེགས་པའི་མདོར་གཏོགས་པ་རྣམས་ཀྱིས་ཆོས་དང་འཐུན་པ་དང། འདུལ་བ་དང་འཐུན་པར་སྨྲས་པ་ན། བདག་ཉིད་བརྗོད་པར་བྱ་བ་མ་ཡིན་པར་མ་བྱེད་པར་ཚེ་དང་ལྡན་པས་བདག་ཉིད་བརྗོད་པར་བྱ་བ་ཁོ་ནར་གྱིས་ཤིག །དགེ་སློང་རྣམས་ཚེ་དང་ལྡན་པ་ལ་ཆོས་དང་འཐུན་པ་དང། འདུལ་བ་དང་འཐུན་པར་སྨྲས་པ་ན། ཚེ་དང་ལྡན་པ་ ཡང་དགེ་སློང་རྣམས་ལ་ཆོས་དང་འཐུན་པ་དང། འདུལ་བ་དང་འཐུན་པར་སྨྲོས་ཤིག །དེ་ཅིའི་ཕྱིར་ཞེ་ན། འདི་ལྟ་སྟེ། གཅིག་ལ་གཅིག་བརྗོད་པར་བྱ་བ་ཉིད་དུ་བྱེད་པ་དང། གཅིག་ལ་གཅིག་འདོམས་ཤིང་རྗེས་སུ་སྟོན་པ་དང། གཅིག་གིས་གཅིག་ལྟུང་བ་ལས་སློང་བ་ཉིད་དེ་ལྟ་བུས་བཅོམ་ལྡན་འདས་དེ་བཞིན་གཤེགས་པ་དགྲ་བཅོམ་པ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་དེའི་འཁོར་འདི་འཕེལ་བར་འགྱུར་གྱིས། ཚེ་དང་ལྡན་པ་ཁྱོད་བདག་ཉིད་བརྗོད་པར་བྱ་བ་མ་ཡིན་པར་བྱེད་པའི་ལས་ཀྱི་མཐའ་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་སློང་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་བསྒོ་བ་ན་གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱ། ཡང་དག་པར་བསྟན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་དགེ་འདུན་ལྷག་མའོ། །
# f4.d འདོན་པའི་མཇུག་བསྡུ་བ་ནི།
ཚེ་དང་ལྡན་པ་དག་བདག་གིས་དགེ་འདུན་ལྷག་མའི་ཆོས་བཅུ་གསུམ་པོ་དག་བཏོན་ཟིན་ཏོ། །
# f4.e རྟོགས་པར་སླ་བའི་དོན་དུ་དོན་བསྡུ་བ་ནི།
དེ་དག་ལས་དགུ་ནི་ལྟུང་བ་དང་པོ་དག་ཉིད་ཀྱིས་སོ། །བཞི་ནི་ལན་གསུམ་གྱི་བར་གྱིས་སོ། །
# f4.f ལྟུང་བ་དག་པའི་དོན་དུ་ཕྱིར་བཅོས་པ་ནི།
དགེ་སློང་གིས་དེ་དག་ལས་ལྟུང་བ་གང་ཡང་རུང་བ་ཞིག་བྱས་ལ། ཇི་སྲིད་དུ་ཤེས་བཞིན་དུ་འཆབ་པར་བྱེད་པ་དེ་སྲིད་དུ་དེས་མི་འདོད་བཞིན་དུ་སྤོ་བར་བྱའོ། །དགེ་སློང་གིས་མི་འདོད་བཞིན་དུ་སྤོ་བ་བྱས་ནས་དེའི་འོག་ཏུ་ཞག་དྲུག་གི་བར་དུ་དགེ་འདུན་མགུ་བར་བྱ་བ་སྤྱད་པར་བྱ་བའོ། །དགེ་སློང་གིས་མགུ་བར་བྱ་བ་སྤྱད་ནས་དབྱུང་བར་བྱ་བ་ལ་ཐོགས་ཏེ་ཆོས་དང་འཐུན་པར་བྱས་ལ། དགེ་སློང་གིས་དགེ་འདུན་གྱི་སེམས་མགུ་བར་བྱས་ན་གང་ན་དགེ་སློང་ཉི་ ཤུའི་ཚོགས་ཀྱི་དགེ་འདུན་ཡོད་པ་དེར་དགེ་སློང་དེ་དབྱུང་བར་བྱའོ། །གལ་ཏེ་དགེ་སློང་ཉི་ཤུར་གཅིག་གིས་མ་ཚང་བའི་ཚོགས་ཀྱི་དགེ་འདུན་གྱིས་དགེ་སློང་དེ་འབྱིན་པར་བྱེད་ན། དགེ་སློང་དེ་ཡང་མ་ཕྱིན་ལ་དགེ་སློང་དེ་དག་ཀྱང་སྨད་པར་བྱ་བ་ཡིན་ ཏེ་དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f4.g རྣམ་པར་བཞག་པ་བསྒྲག་པ་ནི།
དེ་ལ་བདག་གིས་ཚེ་དང་ལྡན་པ་དག་ལ་ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་དྲིའོ། །ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་ལན་གཉིས་ལན་གསུམ་དུ་དྲིའོ། །འདི་ལ་ཚེ་དང་ལྡན་པ་དག་ཡོངས་སུ་དག་ན་འདི་ལྟར་ཅང་མི་སྨྲ་བས་དེ་དེ་བཞིན་དུ་འཛིན་ཏོ། །
# f5 གསུམ་པ་མ་ངེས་པ་འདོན་པ་ལ།
# f5.a སྒྲོ་སྐུར་སྤང་བའི་ཕྱིར་སྡོམ་ནི།
སྡོམ་ལ། དབེན་པ་སྐྱབས་ཡོད་འདུག་པའོ། །
# f5.b ལུང་ལ་གུས་པ་བསྐྱེད་ཅིང་ལུང་འབོག་པ་ནི།
ཚེ་དང་ལྡན་པ་དག་མ་ངེས་པའི་ཆོས་གཉིས་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# f5.c བཅས་པའི་སྐབས་ནི།
ཡང་དགེ་སློང་གང་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་གཅིག་པུ་གཅིག་དང་དབེན་པ་སྐྱབས་ཡོད་པ་ན་འདོད་པ་བྱར་རུང་བ་སྟན་ལ་འདུག་པར་བྱེད་ཅིང། དེ་ལ་གལ་ཏེ་དགེ་བསྙེན་མ་ཡིད་ཆེས་པའི་ཚིག་དང་ལྡན་པས་ཆོས་གསུམ་པོ་ཕམ་པར་འགྱུར་པ་འམ། དགེ་འདུན་ལྷག་མ་འམ། ལྟུང་བྱེད་ལས་ཆོས་གང་ཡང་རུང་བས་སྨྲས་པར་གྱུར་ལ། དགེ་སློང་གིས་འདུག་པར་ཁས་བླངས་ན་ཆོས་གསུམ་པོ་ཕམ་པར་གྱུར་པ་འམ། དགེ་འདུན་གྱི་ལྷག་མ་འམ། ལྟུང་བྱེད་ལས་ཆོས་གང་ཡང་རུང་བས་བྱེད་དུ་གཞུག་ཅིང། དགེ་སློང་དེ་ལ་དགེ་བསྙེན་མ་ཡིད་ཆེས་པའི་ཚིག་དང་ལྡན་པས་ཆོས་གང་དང་གང་གིས་སྨྲས་པར་གྱུར་པའི་ཆོས་དེ་དང་དེས་དགེ་སློང་དེ་བྱེད་དུ་གཞུག་སྟེ། དེ་ནི་མ་ངེས་པའི་ཆོས་སོ། །ཡང་དགེ་སློང་གང་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་གཅིག་པུ་གཅིག་དང་དབེན་པ་སྐྱབས་ཡོད་པ་ན་འདོད་པ་བྱར་མི་རུང་བར་སྟན་ལ་འདུག་པར་བྱེད་ཅིང། དེ་ལ་གལ་ཏེ་དགེ་བསྙེན་མ་ཡིད་ཆེས་པའི་ཚིག་དང་ལྡན་པས་ཆོས་གཉིས་པོ་དགེ་འདུན་ལྷག་མ་འམ། ལྟུང་བྱེད་ལས་ཆོས་གང་ཡང་རུང་བས་སྨྲས་པར་གྱུར་ལ། དགེ་སློང་གིས་འདུག་པར་ཁས་བླངས་ན་ཆོས་གཉིས་པོ་དགེ་འདུན་ལྷག་མ་འམ། ལྟུང་བྱེད་ལས་ཆོས་གང་ཡང་རུང་བས་བྱེད་དུ་གཞུག་ཅིང། དགེ་སློང་དེ་ལ་དགེ་བསྙེན་མ་ཡིད་ཆེས་པའི་ཚིག་དང་ལྡན་པས་ཆོས་གང་དང་གང་གིས་སྨྲས་པར་གྱུར་པའི་ཆོས་དེ་དང་དེས་དགེ་སློང་དེ་བྱེད་དུ་གཞུག་སྟེ། དེ་ཡང་མ་ངེས་པའི་ཆོས་སོ། །
# f5.d འདོན་པ་འཇུག་བསྡུ་བ་ནི།
ཚེ་དང་ལྡན་པ་དག་བདག་གིས་མ་ངེས་པའི་ཆོས་གཉིས་པོ་དག་བཏོན་ཟིན་ཏོ། །
# f5.e དགེ་བ་རྣམ་པར་བཞག་པ་བསྒྲག་པ་ནི།
དེ་ལ་བདག་གིས་ཚེ་དང་ལྡན་པ་དག་ལ་ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་དྲིའོ། །ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་ལན་གཉིས་ལན་གསུམ་དུ་དྲིའོ། །འདི་ལ་ཚེ་དང་ལྡན་པ་དག་ཡོངས་སུ་དག་ན་འདི་ལྟར་ཅང་མི་སྨྲ་བས་དེ་དེ་བཞིན་དུ་འཛིན་ཏོ། །
# f6 རིམ་གྱིས་འདོན་པ། ༼རིལ་གྱིས་འདོན་པ།༽
# f6.1 སྤང་ལྟུང་གི་སྡེ།
# f6.1.a སྒྲོ་འདོགས་སྐུར་འདེབས་སྤོང་བའི་དོན་དུ་སྡོམ་ལ།
སྡོམ་ལ། འཆང་བ་འབྲལ་བ་འཇོག་པ་དང། །འཁྲུར་འཇུག་པ་དང་ལེན་པ་དང། །སློང་དང་སྟོད་གཡོགས་སྨད་གཡོགས་བཅས། །རིན་ཐང་སོ་སོར་བསྐུར་བ་འོ། །
# f6.1.b ལུང་འབོག་པ་ནི།
ཚེ་དང་ལྡན་པ་དག་སྤང་བའི་ལྟུང་བྱེད་ཀྱི་ཆོས་སུམ་བཅུ་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# f6.1.c.1.1 འཆང་བ་ལ།
དགེ་སློང་ཆོས་གོས་ཟིན་པས་སྲ་བརྐྱང་ཕྱུང་ན། ཞག་བཅུའི་བར་དུ་གོས་ལྷག་པ་རྣམ་པར་མ་བརྟགས་པ་བཅང་བར་བྱའོ། །དེ་ལས་འདས་པར་བཅངས་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.2 གཉིས་པ་འབྲལ་བ་ནི།
དགེ་སློང་ཆོས་གོས་ཟིན་པས་སྲ་བརྐྱང་ཕྱུང་ན། གལ་ཏེ་ནུབ་གཅིག་ཀྱང་ཆོས་གོས་གསུམ་ལས་ཆོས་གོས་གང་ཡང་རུང་བ་དང་མཚམས་ཀྱི་ཕྱི་རོལ་དུ་འབྲལ་བར་བྱེད་ན། དགེ་འདུན་གྱིས་གནང་བ་མ་གཏོགས་ཏེ་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.3 གསུམ་པ་ཟླ་འཇོག་ནི།
དགེ་སློང་ཆོས་གོས་ཟིན་པས་སྲ་བརྐྱང་ཕྱུང་ན། དུས་མ་ཡིན་པའི་གོས་ཤིག་རྙེད་ལ་འདོད་ན་དགེ་སློང་དེས་གོས་དེ་བླང་བར་བྱའོ། །བླངས་ནས་གལ་ཏེ་ལངས་ན་མྱུར་བ་ཁོ་ནར་གོས་སུ་བྱས་ལ་བཅང་བར་བྱའོ། །གལ་ཏེ་མི་ལངས་ལ་ཁ་མ་ལངས་པ་ཁ་སྐོང་བའི་གོས་ལ་རེ་བ་ཡོད་ན་དགེ་སློང་དེས་གོས་དེ་ཟླ་བ་གཅིག་གི་མཐར་གཞག་པར་བྱའོ། །དེ་ལས་འདས་པར་འཇོག་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.4 འཁྲུར་འཇུག་པ་ལ།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པ་ལ་གོས་རྙིང་པ་འཁྲུར་འཇུག་གམ། འཚེད་དུ་འཇུག་གམ། འཆག་ཏུ་འཇུག་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.5+6?? ལོན་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པ་ལས་གོས་ལེན་ན། བརྗེས་པ་མ་གཏོགས་ཏེ་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.7 བདུན་པ་སྟོད་གཡོགས་སྨད་གཡོགས་དང་བཅས།ཞེས་སློང་བར་མི་རིགས་པ་སློང་བའི་སྤང་བ་ལ།
ཡང་དགེ་སློང་གང་ཁྱིམ་བདག་གམ། ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པའི་གན་དུ་སོང་སྟེ། གོས་སློང་ན་དུས་མ་གཏོགས་ཏེ། སྤང་བའི་ལྟུང་བྱེད་དོ། །དེ་ལ་དུས་ནི་འདི་ཡིན་ཏེ། དགེ་སློང་གོས་ཕྲོགས་སམ། གོས་རླག་གམ། གོས་ཚིག་གམ། གོས་རླུང་གིས་ཁྱེར་རམ། གོས་ཆུས་ཁྱེར་ན་དེ་ལ་དེ་ནི་དུས་ཡིན་ནོ། །

# f6.1.c.1.8 བརྒྱད་པ་དཔོགས་པ་སློང་བ་ནི།
དགེ་སློང་གི་ཕྱིར་ཁྱིམ་བདག་གམ། ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པས་གོས་ཀྱི་རིན་དག་སྤགས་ནས་བདག་གི་གོས་ཀྱི་རིན་འདི་དག་གིས་གོས་འདི་དང་འདི་ལྟ་བུ་ཞིག་ཉོས་ལ། དགེ་སློང་མིང་འདི་ཞེས་བྱ་བ་འོངས་ན། དེ་ལ་གོས་རུང་བ་དུས་སུ་ བསྐོན་ནོ་སྙམ་པ་ལས། དེ་ལ་གལ་ཏེ་དགེ་སློང་དེ་ལ་སྔར་མ་བསྟབས་པར་རྣམ་པར་རྟོགས་པ་འགའ་ཞིག་བྱུང་སྟེ། བཟང་པོ་འདོད་པའི་ཕྱིར་ཁྱིམ་བདག་གམ། ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པ་དེའི་གན་དུ་སོང་ནས་འདི་སྐད་ཅེས་ཚེ་དང་ལྡན་པས་བདག་གི་ཕྱིར་གོས་ཀྱི་རིན་གང་དག་སྤགས་པའི་གོས་ཀྱི་རིན་དེ་དག་གིས་ཚེ་དང་ལྡན་པ་གོས་འདི་དང་འདི་ལྟ་བུ་ཞིག་ཉོས་ལ་བདག་ལ་གོས་རུང་བ་དུས་སུ་ལེགས་པར་སྐོན་ཅིག་ཅེས་སྨྲས་ཏེ། གོས་གྲུབ་པ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.9 དགུ་པ་སོ་སོ་ནས་དཔགས་པ་སློང་བ་ནི།
དགེ་སློང་གི་ཕྱིར་ཁྱིམ་བདག་དང་ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པས་གོས་ཀྱི་རིན་སོ་སོ་དེ་དག་སྤགས་ནས། བདག་ཅག་གཉིས་ཀྱིས་གོས་ཀྱི་རིན་སོ་སོ་བ་འདི་དག་གིས་གོས་འདི་དང་འདི་ལྟ་བུ་སོ་སོ་བ་དག་ཉོས་ལ། དགེ་སློང་མིང་འདི་ཞེས་བྱ་བ་འོངས་པ་དང། བདག་ཅག་གཉིས་ཀྱིས་སོ་སོ་ནས་དེ་ལ་གོས་རུང་བ་དུས་སུ་བསྐོན་ནོ། །སྙམ་པ་ལས་དེ་ན་གལ་ཏེ་དགེ་སློང་དེ་ལ་སྔར་མ་བསྟབས་པར་རྣམ་པར་རྟོག་པ་འགའ་ཞིག་བྱུང་སྟེ། བཟང་པོ་འདོད་པའི་ཕྱིར་ཁྱིམ་བདག་དང། ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པ་དེ་དག་གི་གན་དུ་སོང་ནས་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་དག་གིས་བདག་གི་ཕྱིར་གོས་ཀྱི་རིན་སོ་སོ་བ་དག་སྤགས་པའི་གོས་ཀྱི་རིན་སོ་སོ་བ་དེ་དག་གིས་ཚེ་དང་ལྡན་པ་དག་གོས་འདི་དང་འདི་ལྟ་བུ་སོ་སོ་བ་དག་ཉོས་ལ་གཉི་ག་འདུས་ཏེ། བདག་ལ་གོས་རུང་བ་ཅིག་དུས་སུ་ལེགས་པར་སྐོན་ཅིག་ཅེས་སྨྲས་ཏེ། གོས་གྲུབ་པ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.1.10 བཅུ་པ་བསྐུར་བའི་སྤང་བ་ནི།
དགེ་སློང་གི་ཕྱིར་རྒྱལ་པོ་འམ། བློན་པོ་ཆེན་པོ་འམ། བྲམ་ཟེ་འམ། ཁྱིམ་བདག་གམ། གྲོང་རྡལ་གྱི་མི་འམ། ཡུལ་མི་འམ། ནོར་ཅན་ནམ། ཚོང་དཔོན་ནམ། དེད་དཔོན་གྱིས་ཕོ་ཉའི་ལག་ཏུ་གོས་ཀྱི་རིན་དག་བསྐུར་བར་གྱུར་ལ། དེ་ནས་ཕོ་ཉ་དེ་གོས་ཀྱི་རིན་དེ་དག་ཁྱེར་ནས་དགེ་སློང་དེ་ག་ལ་བ་དེར་སོང་སྟེ་ཕྱིན་ནས་དགེ་སློང་དེ་ལ་འདི་སྐད་ཅེས། འཕགས་པ་མཁྱེན་པར་མཛོད་ཅིག །ཁྱོད་ལ་རྒྱལ་པོ་འམ། བློན་པོ་ཆེན་པོ་འམ། བྲམ་ཟེ་འམ། ཁྱིམ་བདག་གམ། གྲོང་རྡལ་གྱི་མི་འམ། ཡུལ་མི་འམ། ནོར་ཅན་ནམ། ཚོང་དཔོན་ནམ། དེད་དཔོན་ཆེ་གེ་མོས་གོས་ཀྱི་རིན་འདི་དག་བསྐུར་གྱིས། འཕགས་པས་ཐུགས་བརྩེ་བའི་སླད་དུ་འདི་དག་བཞེས་ཤིག་ཅེས་ཟེར་ན་དགེ་སློང་དེས་ཕོ་ཉ་དེ་ལ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཕོ་ཉ་དགེ་སློང་དག་ནི་གོས་ཀྱི་རིན་དག་ལེན་དུ་མི་རུང་སྟེ། བདག་ཅག་གོས་རུང་བ་དུས་སུ་རྙེད་ན་ནི་ལེན་ཏོ་ཞེས་བརྗོད་པར་བྱའོ། །གལ་ཏེ་ཕོ་ཉ་དེས་དགེ་སློང་དེ་ལ་འདི་སྐད་ཅེས། འཕགས་པ་རྣམས་ཀྱི་ཞལ་ཏ་བགྱིད་པ་འཕགས་པ་རྣམས་ཀྱི་ཞལ་ཏ་ཉམས་སུ་ལེན་པ་གང་ལགས་པ་འགའ་མཆིས་སམ་ཞེས་ཟེར་ན། དགེ་སློང་གོས་འདོད་པས་ཕོ་ཉ་ལ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཕོ་ཉ་འདི་ནི་དགེ་སློང་རྣམས་ཀྱི་ཞལ་ཏ་བྱེད་པ་སྟེ། འདི་ནི་དགེ་སློང་རྣམས་ཀྱི་ཞལ་ཏ་ཉམས་སུ་ལེན་པའོ་ཞེས་ཀུན་དགའི་ར་བ་པ་འམ། དགེ་བསྙེན་ཞལ་ཏ་བྱེད་པ་བསྟན་པར་བྱའོ། །དེ་ནས་ཕོ་ཉ་དེས་གོས་ཀྱི་རིན་དེ་དག་ཁྱེར་ནས་ཞལ་ཏ་བྱེད་པ་ག་ལ་བ་དེར་འགྲོ་བར་བྱ་ཞིང། ཕྱིན་ནས་ཞལ་ཏ་བྱེད་པ་ལ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཞལ་ཏ་བྱེད་པ་ཤེས་པར་གྱིས་ཤིག །ཁྱོད་ཀྱིས་གོས་ཀྱི་རིན་འདི་དག་གིས་གོས་འདི་དང་འདི་ལྟ་བུ་ཞིག་ཉོས་ལ་དགེ་སློང་མིང་འདི་ཞེས་བྱ་བ་འོངས་པ་དང། དེ་ལ་གོས་རུང་བ་དུས་སུ་སྐོན་ཅིག་ཅེས་བརྗོད་པར་བྱའོ། །དེ་ནས་ཕོ་ཉ་དེས་ཞལ་ཏ་བྱེད་པ་དེ་ལ་ཤིན་ཏུ་ལེགས་པར་ཡང་དག་པར་བསྒོ་ཞིང་། ཡང་དག་པར་བསྟན་ནས། དགེ་སློང་དེ་ག་ལ་བ་དེར་འགྲོ་བར་བྱ་ཞིང་ཕྱིན་ནས་དགེ་སློང་དེ་ལ་འདི་སྐད་ཅེས། འཕགས་པས་ཞལ་ཏ་བགྱིད་པ་བསྟན་པ་གང་ལགས་པ་དེ་ལ་བདག་གིས་ཡང་དག་པར་བསྟན་ལགས་ཀྱིས་དེའི་གན་དུ་བཞུད་ཅིག་དང། དེས་ཁྱོད་ལ་གོས་རུང་བ་དུས་སུ་བསྐོན་པར་འགྱུར་རོ་ཞེས་བརྗོད་པར་བྱའོ། །དགེ་སློང་གོས་འདོད་པས་ཞལ་ཏ་བྱེད་པའི་གན་དུ་སོང་ནས་ཚེ་དང་ལྡན་པ་ཞལ་ཏ་བྱེད་པ་བདག་གོས་འདོད་དོ། །ཚེ་དང་ལྡན་པ་ཞལ་ཏ་བྱེད་པ་བདག་གོས་འདོད་དོ་ཞེས་ལན་གཉིས་ལན་གསུམ་དུ་བསྐུལ་བར་བྱ། དྲན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་བསྐུལ་ཞིང་དྲན་པར་ བྱས་པ་ན་གལ་ཏེ་གོས་དེ་གྲུབ་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མ་གྲུབ་ན་ལན་བཞི་ལན་ལྔ་ལན་དྲུག་གི་བར་དུ་ཕྱོགས་སུ་ཅང་མི་སྨྲ་བར་བསྡད་པར་བྱའོ། །ལན་བཞི་ལན་ལྔ་ལན་དྲུག་གི་བར་དུ་ཕྱོགས་སུ་ཅང་མི་སྨྲ་བར་བསྡད་པ་ན། གལ་ཏེ་གོས་དེ་གྲུབ་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མ་གྲུབ་ན་དེའི་འོག་ཏུ་གོས་བསྒྲུབ་པའི་ཕྱིར་བརྩལ་ཏེ་གོས་གྲུབ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །གལ་ཏེ་མ་གྲུབ་ན་ཕྱོགས་གང་ནས་གོས་ཀྱི་རིན་དེ་དག་འོངས་པ་དེར་བདག་འགྲོ་བར་བྱའོ། །ཡང་ན་ཡིད་བརྟན་པའི་ཕོ་ཉ་ལས་ཚེ་དང་ལྡན་པ་དག་གིས་དགེ་སློང་ཆེ་གེ་མོའི་ཕྱིར་གོས་ཀྱི་རིན་གང་དག་བསྐུར་བ་དེ་དག་ནི་དགེ་སློང་དེའི་དོན་ཅིར་ཡང་མ་གྲུབ་ཀྱིས་ཤེས་པར་གྱིས་ཤིག །ཚེ་དང་ལྡན་པ་དག་གིས་རང་གི་ནོར་ཆུད་མ་གསན་ཅིག་ཅེས་སྤྲིང་བར་བྱ་སྟེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f6.1.c.2 བཅུ་ཚན་གཉིས་པ་སྟན་གྱི་སྡེ་ནི།
སྡོམ་ལ། སྲིན་བལ་འབའ་ཞིག་ཆ་གཉིས་དང། །དྲུག་དང་མཐོ་གང་ལམ་དང་ནི། །འཁྲུ་བ་དང་ནི་གསེར་དངུལ་དང། །མངོན་མཚན་ཅན་དང་ཉོ་ཚོང་ངོ། །
# f6.1.c.2.1 སྲིན་བལ་འབའ་ཞིག་པ་ལ།
ཡང་དགེ་སློང་གང་སྲིན་བལ་གྱི་སྟན་སར་པ་བྱེད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །ཡང་དགེ་སློང་གང་ལུག་བལ་ནག་པོ་འབའ་ཞིག་གི་སྟན་སར་པ་བྱེད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.2 གཉིས་པ་ནག་པོ་འབའ་ཞིག་པ་ནི།
དགེ་སློང་གིས་སྟན་སར་པ་བྱེད་ན་ཆ་གཉིས་ནི་ལུག་བལ་ནག་པོ་འབའ་ཞིག་ལས་གཞུག་པར་བྱ། གསུམ་པ་ནི་དཀར་པོ་ལས། བཞི་པ་ནི་འཁོབ་བལ་ལས་གཞུག་པར་བྱའོ། །གལ་ཏེ་དགེ་སློང་གིས་ཆ་གཉིས་ལུག་བལ་ནག་པོ་འབའ་ཞིག་ལས་མ་བཅུག་གམ། གསུམ་པ་དཀར་པོ་ལས། བཞི་པ་ནི་འཁོབ་བལ་ལས་མ་བཅུག་པར་སྟན་སར་པ་བྱེད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.3 གསུམ་པ་ཆ་གཉིས་བསྲེ་བ་ལ།
དགེ་སློང་གིས་སྟན་སར་པ་བྱེད་ན་མི་འདོད་བཞིན་དུ་ལོ་དྲུག་ཏུ་བཅང་བར་བྱའོ། །གལ་ཏེ་དགེ་སློང་ལོ་དྲུག་ཚུན་ཆད་དུ་སྟན་རྙིང་པ་དེ་སྤངས་ཀྱང་རུང། མ་སྤངས་ཀྱང་རུང། སྟན་སར་པ་གཞན་བྱེད་ན་དགེ་འདུན་གྱིས་གནང་བ་མ་གཏོགས་ཏེ་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.4 བཞི་པ་ལོ་དྲུག་གི་ནང་དུ་སྟན་གཉིས་པ་བྱེད་པ་ནི།
དགེ་སློང་གིས་གདིང་བ་སར་པ་བྱེད་ན་སར་པ་ཁ་དོག་མི་སྡུག་པར་བྱ་བའི་ཕྱིར་གདིང་བ་རྙིང་པ་བདེ་བར་གཤེགས་པའི་མཐོ་གང་འཁོར་བས་ཀླན་པར་བྱའོ། །
# f6.1.c.2.5 དུས་ལས་འདས་པའི་ལྟུང་བ་ནི།
གལ་ཏེ་དགེ་སློང་གིས་གདིང་བ་སར་པར་ཁ་དོག་མི་སྡུག་པར་བྱ་བའི་ཕྱིར། གདིང་བ་རྙིང་པ་བདེ་བར་གཤེགས་པའི་མཐོ་གང་འཁོར་བས་མ་ཀླན་པར་གདིང་བ་སར་པ་སྤྱོད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.6 དྲུག་པ་བལ་འཁུར་པ་ལས་གྱུར་པའི་སྤང་བ་ནི།
དགེ་སློང་ལམ་དུ་ཞུགས་པས་ལུག་བལ་དག་རྙེད་ལ་འདོད་ན་དགེ་སློང་དེས་བླང་བར་བྱའོ། །བླངས་ནས་ཁྱེར་བ་མེད་ན་དཔག་ཚད་གསུམ་གྱི་མཐའི་བར་དུ་བདག་ཉིད་ཀྱིས་བསྐུར་བར་བྱའོ། །དེ་ལས་འདས་པར་ཁྱེར་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.7 བདུན་པ་བལ་རྨེལ་དུ་འཇུག་པའི་སྤང་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པ་ལ་ལུག་བལ་འཁྲུར་འཇུག་གམ། འཚེད་དུ་འཇུག་གམ། རྨེལ་དུ་འཇུག་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.8 བརྒྱད་པ་རིན་པོ་ཆེ་བདག་གིར་བྱེད་པའི་སྤང་བ་ནི།
ཡང་དགེ་སློང་གང་རང་གི་ལག་གིས་གསེར་དང་། དངུལ་ལེན་ཏམ། གཞན་ལེན་དུ་འཇུག་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.9 དགུ་པ་མངོན་མཚན་ཅན་ལ།
ཡང་དགེ་སློང་གང་མངོན་མཚན་ཅན་གྱི་སྤྱོད་པ་རྣམ་པ་སྣ་ཚོགས་བྱེད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.2.10 བཅུ་པ་ཉོ་ཚོང་ལ་
ཡང་དགེ་སློང་གང་ཉོ་ཚོང་རྣམ་པ་སྣ་ཚོགས་བྱེད་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3 ལྷུང་བཟེད་ཀྱི་སྡེ།
སྡོམ་ལ། ལྷུང་བཟེད་གཉིས་དང་ཐ་ག་གཉིས། །བྱིན་ཕྲོགས་སྟོན་ཟླ་ཐ་ཆུངས་དང། །དགོན་པ་པ་དང་རས་ཆེན་དང། །བསྔོས་པ་དང་ནི་གསོག་འཇོག་གོ །
# f6.1.c.3.1 རྟེན་བཅང་དུ་གནང་བ་ནི།
ཡང་དགེ་སློང་གིས་ལྷུང་བཟེད་ལྷག་མ་ཞག་བཅུར་བཅང་བར་བྱའོ། །དེ་ལས་འདས་པར་འཆང་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.2 ལྷུང་བཟེད་ལྷག་པོ་ཚོལ་བའི་སྤང་བ་ནི།
ཡང་དགེ་སློང་གང་ལྷུང་བཟེད་ལྷན་པ་ལྔ་མེད་པ་སྤྱད་བཟོད་པ་ཡོད་བཞིན་དུ་བཟང་པོ་འདོད་པའི་ཕྱིར་ལྷུང་བཟེད་སར་པ་གཞན་ཚོལ་ཞིང་ལྷུང་བཟེད་གྲུབ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །དགེ་སློང་དེས་ལྷུང་བཟེད་དེ་དགེ་སློང་གི་འཁོར་ལ་དབུལ་བར་བྱའོ། །དགེ་སློང་གི་འཁོར་དེའི་ལྷུང་བཟེད་ཐ་མར་གྱུར་པ་གང་ཡིན་པ་དེ་དགེ་སློང་དེ་ལ་དགེ་སློང་ཁྱོད་ཀྱིས་ལྷུང་བཟེད་འདི་བྱིན་གྱིས་བརླབ་པར་མི་བྱ། གཏང་བ་མི་བྱ། གཞན་ལ་སྦྱིན་པར་མི་བྱ་བར་ཆག་པའི་མཐར་ཐུག་གི་བར་དུ་ཁད་ཀྱིས་དལ་བུས་དལ་བུས་སྤྱད་པར་བྱའོ་ཞེས་སྦྱིན་པར་བྱ་སྟེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f6.1.c.3.3 འཐག་ཏུ་འཇུག་པའི་སྤང་བ་ནི།
ཡང་དགེ་སློང་གང་རང་གི་ལག་གིས་བསླངས་པའི་དོག་པ་ཐ་ག་པ་ཉེ་དུ་མ་ཡིན་པ་ལ་འཐག་ཏུ་འཇུག་ན་གོས་གྲུབ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.4 བཞི་པ་འཐག་པ་བསྐྱེད་པ་ནི།
དགེ་སློང་གི་ཕྱིར་ཁྱིམ་བདག་གམ་ཁྱིམ་བདག་གི་ཆུང་མ་ཉེ་དུ་མ་ཡིན་པས་ཐ་ག་པ་ཉེ་དུ་མ་ཡིན་པ་ལ་གོས་འཐག་ཏུ་བཅུག་པ་ལས། དེ་ལ་གལ་ཏེ་དགེ་སློང་དེ་ལ་སྔར་མ་བསྟབས་པར་རྣམ་པར་རྟོག་པ་འགའ་ཞིག་བྱུང་ནས་ཐ་ག་པ་ཉེ་དུ་མ་ཡིན་པའི་ གན་དུ་སོང་སྟེ་འདི་སྐད་ཅེས། ཚེ་དང་ལྡན་པ་ཐ་ག་པ་ཤེས་པར་གྱིས་ཤིག །གོས་འདི་ནི་ཁོ་བོའི་ཕྱིར་འཐག་གིས། ཚེ་དང་ལྡན་པ་ཐ་ག་པ་གོས་འདི་ཞེང་ཆེ་བ་དང། ཁྲུན་རིང་བ་དང། ཐར་གྱིས་ཞག་པ་དང། ཐགས་རན་བཟང་བར་ ལེགས་པར་གྱིས་ཤིག །ཚེ་དང་ལྡན་པ་ཐ་ག་པ་འདི་ལྟར་ཁོ་བོས་གླ་རྔན་འདི་ལྟ་སྟེ། བཟའ་བ་དང་། བཅའ་བ་ཙམ་འམ། བཟའ་རྒྱུ་ཅུང་ཟད་ཅིག་སྦྱིན་ནོ་ཞེས་སྨྲས་ནས། དེ་ལ་གལ་ཏེ་དགེ་སློང་དེས་གོས་བསྒྲུབ་པའི་ཕྱིར་གླ་རྔན་འདི་ལྟ་སྟེ། བཟའ་བ་འམ། བཅའ་བ་ཙམ་འམ། བཟའ་རྒྱུ་ཅུང་ཟད་ཅིག་སྦྱིན་ཞིང་གོས་གྲུབ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.5 ལྔ་པ་བྱིན་འཕྲོག་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་ལ་གོས་བྱིན་ནས་དེའི་འོག་ཏུ་ཁྲོས་འཁྲུགས་རྔམས་པར་གྱུར་ཏེ། ཡིད་མ་རངས་ནས་འཕྲོག་གམ། འཕྲོག་ཏུ་འཇུག་ཅིང་དེ་ལ་འདི་སྐད་ཅེས། དགེ་སློང་ཁྱོད་ལ་གོས་མི་སྦྱིན་གྱིས། ཕྱིར་བྱིན་ཅིག་ཅེས་ཟེར་ན་དགེ་སློང་དེས་ལྷག་མ་ཡོད་པ་ཕྱིར་སྦྱིན་པར་བྱ་ཞིང་། བཏང་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.6 གཏད་པ་ལས་བྱུང་བའི་སྤང་བ་ནི།
དགེ་སློང་གིས་སྟོན་ཟླ་ཐ་ཆུངས་ཉར་ཞག་བཅུས་མ་ཚང་བ་ཚུན་ཆད་དུ་བརྟད་པ་ལས་བྱུང་བའི་གོས་ཤིག་རྙེད་ལ་འདོད་ན་དགེ་སློང་དེས་གོས་དེ་བསླང་བར་བྱའོ། །བླངས་ནས་གོས་སྦྱིན་པའི་དུས་ཀྱི་བར་དུ་བཅང་བར་བྱའོ། །དེ་ལས་འདས་པར་བཅང་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.7 དགོན་པའི་སྤང་བ་ནི།
དགེ་སློང་རབ་ཏུ་མང་པོ་དག་གནས་མལ་དགོན་པ་དོགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་ཐ་དད་པས་འཇིགས་སུ་རུང་བ་དང་བཅས་པར་གྲགས་པ་དག་ཏུ་དབྱར་ཕྱི་མར་གྱུར་ལ། དགེ་སློང་དགོན་པ་པས་འདོད་ན་ཆོས་གོས་གསུམ་ལ། ཆོས་གོས་གང་ཡང་རུང་བ་ཁྱིམ་གཞན་དུ་གཞག་པར་བྱའོ། །དགེ་སློང་དགོན་པ་པ་ལ་མཚམས་ཀྱི་ཕྱི་རོལ་ཏུ་འགྲོ་དགོས་པ་དེ་ལྟ་བུའི་རྐྱེན་ཞིག་བྱུང་ན་དགེ་སློང་དགོན་པ་པ་དེས་ཞག་དྲུག་གི་མཐའ་ཚུན་ཆད་དུ་ཆོས་གོས་དེ་དང་མཚམས་ཀྱི་ཕྱི་རོལ་ཏུ་འབྲལ་བར་བྱའོ། །དེ་ལས་འདས་པར་འབྲལ་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.8 རས་ཆེན་བཙལ་སྔས་བཞག་ཕྱིས་ཀྱི་སྤང་བ་ནི།
དགེ་སློང་རྣམས་ཀྱིས་སོ་ཀ་རྣམས་ཀྱི་ཟླ་བ་གཅིག་ལུས་ན་དབྱར་གྱི་གོས་རས་ཆེན་བཙལ་བར་བྱའོ་དབྱར་ཟད་ནས་འོག་ཏུ་ཟླ་བ་ཕྱེད་ཀྱི་བར་དུ་བཅང་བར་བྱའོ་གལ་ཏེ་དགེ་སློང་གིས་སོ་ཀ་རྣམས་ཀྱི་ཟླ་བ་གཅིག་ལུས་པའི་སྔོན་རོལ་ཏུ་དབྱར་གྱི་གོས་རས་ཆེན་ཚོལ་བར་བྱེད་དམ། དབྱར་ཟད་ནས་འོག་ཏུ་ཟླ་བ་ཕྱེད་ལས་ལྷག་པར་འཆང་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །
# f6.1.c.3.9 བསྔོས་བསྒྱུར་གྱི་སྤང་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་འདུན་ལ་བསྔོས་པའི་རྙེད་པ་ཤེས་བཞིན་དུ་གང་ཟག་བདག་ལ་སྒྱུར་དུ་འཇུག་ན་སྤང་བའི་ལྟུང་བྱེད་དོ། །

ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་ཁྲི་འམ། ཁྲིའུ་འམ། སྟན་ནང་ཚངས་ཅན་ནམ། ལ་བ་འམ། སྔས་སམ། གོར་བུ་བླ་ཁབ་མེད་པར་བཏིང་ངམ། གདིང་དུ་བཅུག་ནས་མ་བསྡུས་སམ། སྡུད་དུ་མ་བཅུག་གམ། དགེ་སློང་འཁོད་པ་ལ་མ་བཅོལ་བར་དེ་ནས་སོང་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.2.5 གདིང་བ་ལ།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་གཙུག་ལག་ཁང་དུ་རྩྭའི་གདིང་བ་འམ། ལོ་མའི་གདིང་བ་བཏིང་ངམ། འདིང་དུ་བཅུག་ནས་མ་བསྡུས་སམ། སྡུད་དུ་མ་བཅུག་གམ། དགེ་སློང་འཁོད་པ་ལ་མ་བཅོལ་བར་དེ་ནས་སོང་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.2.6 བསྐྲད་པ་ལ།
ཡང་དགེ་སློང་གང་ཁྲོས་འཁྲུགས་རྔམ་པར་གྱུར་ཏེ་ཡིད་མ་རངས་ནས་དགེ་འདུན་གྱི་གཙུག་ལག་ཁང་ནས་དགེ་སློང་སྐྲོད་དམ། སྐྲོད་དུ་འཇུག་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.2.7 ཕྱིས་གནོན་ལས་འདས་ལྟུང་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་གཙུག་ལག་ཁང་ནས་དགེ་སློང་དག་སྔ་ན་འཁོད་པར་ཤེས་བཞིན་དུ་ཕྱིས་འོངས་ན་སུ་ལ་གནོད་པ་དེ་འགྲོ་བར་འགྱུར་བ་དེ་ཉིད་དུ་བྱས་ཏེ། ཕྱིས་གནོན་བྱས་ཏེ་སྟན་ལ་ཉལ་ལམ་འདུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.2.8 རྩ་བ་དབྱུང་བ་ལ།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་གཙུག་ལག་ཁང་གི་སྟེང་གི་ནམ་མཁའ་ལ་ཐོག་བོར་བར་ཁྲི་འམ། ཁྲིའུ་རྩ་བ་འབྱུང་བར་ཤེས་བཞིན་དུ་ལྕིད་ཀྱིས་ཕབ་སྟེ་ཉལ་ལམ་འདུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.2.9 འདེབས་པ་ལ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་སྲོག་ཆགས་དང་བཅས་པའི་ཆུས་རྩྭ་འམ། ལྕི་བ་འམ། ས་ལ་འདེབས་སམ། འདེབས་སུ་འཇུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.2.10 རིམ་གཉིས་སུ་བརྩིགས་པ་ལ།
ཡང་དགེ་སློང་གིས་གཙུག་ལག་ཁང་ཆེན་པོ་ཞིག་རྩིག་ཏུ་འཇུག་ན་སྒོའི་སྦུབས་དང། གཏན་པ་དང། སྣང་བའི་གནས་ཇི་ཙམ་པའི་འདུ་ཤེས་ཀྱིས་ས་བརྟགས་པ་ནས་བཟུང་སྟེ། ཕག་གུའི་རིམ་པ་འཇིམ་པ་དང་བཅས་པ་གཉིས་སམ་གསུམ་བརྩིག་པར་བྱའོ། །དེ་ལས་ལྷག་པར་རྩིག་ན་ལྟུང་བྱེད་དོ། །

# f6.2.3 བཅུ་ཚན་གསུམ་པ། མ་བསྐོས་པའི་སྡེ།
# f6.2.3.a བཅུ་ཚན་གསུམ་པ་ལ་སྡོམ་ནི།
སྡོམ་ལ། མ་བསྐོས་ཉི་མ་ནུབ་པ་དང། །ཟས་དང་ཆོས་གོས་གཉིས་དག་དང། །དོན་འཐུན་གྲུ་དང་དབེན་པ་གཉིས། །དགེ་སློང་མ་ཡིས་སྦྱོར་བཅུག་པའོ། །
# f6.2.3.1 མ་བསྐོས་པར་སྟོན་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱིས་མ་བསྐོས་པར་དགེ་སློང་མ་ལ་ཆོས་སྟོན་ན། ཆོས་འདི་ལྟ་བུ་དང་ལྡན་པ་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.3.2 ཉི་མ་མ་ནུབ་ཀྱི་བར་དུ་སྟོན་པ་ནི།
དགེ་སློང་གང་དགེ་འདུན་གྱིས་བསྐོས་ཀྱང་ཉི་མ་ནུབ་ཀྱི་བར་དུ་དགེ་སློང་མ་ལ་ཆོས་སྟོན་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.3 ཟས་ཅུང་ཟད་ཀྱི་ཕྱིར་སྟོན་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་རྣམས་ལ་འདི་སྐད་ཅེས་དགེ་སློང་དག་ཟས་ཅུང་ཟད་ཙམ་གྱི་ཕྱིར་དགེ་སློང་མ་ལ་སྟོན་ཏོ། །ཞེས་ཟེར་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.4 གོས་བྱིན་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པ་ལ་གོས་བྱིན་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.5 གོས་བྱེད་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པའི་གོས་བྱེད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.6 དགེ་སློང་མ་དང་ལྷན་གཅིག་ལམ་དུ་འགྲོ་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མའི་དོན་འཐུན་དང་ལྷན་ཅིག་འཐམས་ནས་འདྲོན་ལམ་དུ་འཇུག་ན་དུས་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །དེ་ལ་དུས་ནི་འདི་ཡིན་ཏེ། ལམ་དོན་འཐུན་གྱིས་བགྲོད་པར་བྱ་བ་དོགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་ཐ་དད་པས་འཇིགས་སུ་རུང་བ་དང་བཅས་པར་གྲགས་པ་ཞིག་ན་དེ་ལ་དེ་ནི་དུས་ཡིན་ནོ། །
# f6.2.3.7 དགེ་སློང་མ་དང་ལྷན་གཅིག་གྲུ་ལ་འཇུག་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མའི་དོན་འཐུན་དང་ལྷན་ཅིག་འཐམས་ནས་གྲུ་གཅིག་ཏུ་འཇུག་ཅིང་གྱེན་དུ་འགྲོ་འམ། ཐུར་དུ་འགྲོ་ན་ཐད་ཀར་ཕ་རོལ་ཏུ་འགྲོ་བ་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.3.8 དབེན་པ་ན་འདུག་པ་ལ།
ཡང་དགེ་སློང་གང་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་གཅིག་པུ་གཅིག་དང་དབེན་པ་སྐྱབས་ཡོད་པ་ན་སྟན་ལ་འདུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.9 འགྲེང་བ་ལ།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་དང་ལྷན་ཅིག་གཅིག་པུ་གཅིག་དང་དབེན་པ་སྐྱབས་ཡོད་པ་ན་འགྲེང་ན་ལྟུང་བྱེད་དོ། །
# f6.2.3.10 དགེ་སློང་མ་སྦྱོར་དུ་བཅུག་པ་ནི།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་དགེ་སློང་མས་སྦྱོར་དུ་བཅུག་པའི་ཟས་ཟ་ན་སྔར་ཁྱིམ་དུ་བོས་པ་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.4 བཅུ་ཚན་བཞི་པ། ཡང་ཡང་གི་སྡེ།
# f6.2.4.a བཅུ་ཚན་བཞི་པ། སྡོམ་ནི།
སྡོམ་ལ། ཡང་ཡང་དང་ནི་འདུག་གནས་གཅིག །ཕྱེ་དང་བཅའ་དང་སྟོབས་པ་དང་། །དུས་དང་དུས་མིན་སོགས་འཇོག་དང་། །ཁ་ནས་མིད་དང་བསོད་པ་ཉིད། །
# f6.2.4.1 ཡང་ཡང་ཟ་བ་ལ།
ཡང་ཡང་ཟ་ན་དུས་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །དེ་ལ་དུས་ནི་འདི་ཡིན་ཏེ། ན་བ་དུས་སོ། །ལས་དུས་སོ། །ལམ་དུས་སོ། །གོས་སྦྱིན་པའི་ཚེ་དུས་ཏེ། དེ་ལ་དེ་ནི་དུས་ཡིན་ནོ། །
# f6.2.4.2 འདུག་གནས་གཅིག་ཏུ་ཟ་བ་ལ།
འདུག་གནས་གཅིག་ཏུ་ཞག་ལོན་པའི་དགེ་སློང་མི་ན་བས་བསོད་སྙོམས་གཅིག་བཟའ་བར་བྱའོ། །དེ་ལས་ལྷག་པར་ཟ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.4.3 ལྷུང་བཟེད་དོ་གསུམ་ལས་ལྷག་པར་ལེན་པ་ལ།
དགེ་སློང་རབ་ཏུ་མང་པོ་དག་ཁྱིམ་རྣམས་སུ་དོང་བ་ལས། གལ་ཏེ་དེ་དག་ལ་བྲམ་ཟེ་དང་། ཁྱིམ་བདག་དད་པ་ཅན་དག་གིས་ཕྱེ་དང་ཁུར་བ་དག་དུས་ཀྱིས་སྟོབས་པར་གྱུར་ལ། འདོད་ན་དགེ་སློང་དེ་དག་གིས་ལྷུང་བཟེད་གང་བ་གཉིས་སམ་གསུམ་བླང་བར་བྱའོ། །དེ་ལས་ལྷག་པར་ལེན་ན་ལྟུང་བྱེད་དོ། །

ལྷུང་བཟེད་གང་བ་གཉིས་སམ་གསུམ་བླངས་ནས་ཕྱི་རོལ་ཀུན་དགའ་ར་བར་སོང་ལ། དགེ་སློང་འཁོད་པ་རྣམས་ལ་ཡང་བགོ་བཤའ་བྱ། བདག་ཅག་ཀྱང་བཟའ་བར་བྱ་སྟེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f6.2.4.4 སྤངས་པ་ཟ་བ་ལ་
ཡང་དགེ་སློང་གང་ཟས་ཟོས་ཟིན་ཅིང་སྤངས་པ་ལས་ལྷག་པོར་མ་བྱས་པར་བཅའ་བ་འམ་བཟའ་བ་འཆའ་འམ་ཟ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.4.5 སྤངས་པ་སྟོབ་པ་ལ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་དགེ་སློང་ཟས་ཟོས་ཟིན་ཅིང་སྤངས་པ་ལ་སྐབས་ཚོལ་ཞིང་ཅི་ནས་ཀྱང་དགེ་སློང་འདི་ཉེས་པ་འབྱུང་བར་བྱའོ་སྙམ་པ་དེ་ཉིད་རྐྱེན་དུ་བྱས་ནས། ཚེ་དང་ལྡན་པ་འདི་འཆོ་ཤིག་འདི་ཟོ་ཤིག་ཅེས་ལྷག་པོར་མ་བྱས་པའི་བཅའ་བ་དང་བཟའ་བ་དུས་ཀྱིས་སྟོབས་ན་ལྟུང་བྱེད་དོ། །
# f6.2.4.6 འདུས་ཤིང་ཟ་བ་ལ།
འདུས་ཤིང་ཟ་ན་དུས་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
དེ་ལ་དུས་ནི་འདི་ཡིན་ཏེ། ན་བ་དུས་སོ། །ལས་དུས་སོ། །ལམ་དུས་སོ། །གྲུར་ཞུགས་པ་དང། འདུས་པ་ཆེན་པོ་དང། དགེ་སྦྱོང་གི་ཟས་དུས་ཏེ་དེ་ལ་ནི་དུས་ཡིན་ནོ། །
# f6.2.4.7 དུས་མིན་པར་ཟ་བ་ལ།
ཡང་དགེ་སློང་གང་དུས་མ་ཡིན་པར་བཅའ་བ་འམ་བཟའ་བ་འཆའ་བ་འམ་ཟ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.4.8 གསོག་འཇོག་ཟ་བ།
ཡང་དགེ་སློང་གང་བཅའ་བ་དང་བཟའ་བ་སོགས་འཇོག་བྱས་པ་འཆའ་འམ་ཟ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.4.9 བྱིན་ལེན་མ་བྱས་པར་ཟ་བ་ལ།
ཡང་དགེ་སློང་གང་མ་བྱིན་པར་ཁ་ནས་མིད་པའི་ཟས་ཟ་ན་ཆུ་དང་སོ་ཤིང་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.4.10 བསོད་པ་བླང་བ་ལ།
བཅོམ་ལྡན་འདས་ཀྱིས་དགེ་སློང་རྣམས་ཀྱི་ཟས་བསོད་པ་གང་དག་གསུངས་པ་འདི་ལྟ་སྟེ། འོ་མ་དང། ཞོ་དང། མར་དང། ཉ་ཤ་དང། ཤ་དང་ཤ་སྐམ་དག་སྟེ། དགེ་སློང་མི་ན་བར་བདག་ཉིད་ཀྱི་ཕྱིར་ཟས་བསོད་པ་དེ་ལྟ་བུ་དག་གཞན་གྱི་ཁྱིམ་དག་ནས་བསླངས་ཏེ་འཆའ་འམ་ཟ་ན་ལྟུང་བྱེད་དོ། །

# f6.2.5 བཅུ་ཚན་ལྔ་པ། ཆུ་དང་གི་སྡེ།
# f6.2.5.a སྡོམ་ལ།
སྡོམ་ལ། སྲོག་ཆགས་བཅས་དང་ཉལ་སར་འདུག །འགྲེང་དང་གཅེར་བུ་དམག་དང་ནི། །ཞག་གཉིས་བཤམ་དཀྲུག་འགྲོ་བ་དང། །རྡེག་དང་གཟས་དང་གནས་ངན་ལེན། །
# f6.2.5.1 སྲོག་ཆགས་དང་བཅས་པ་ལ་སྤྱོད་པ་ལ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་སྲོག་ཆག་དང་བཅས་པའི་ཆུ་ལ་སྤྱོད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.2 ཉལ་པོ་བྱེད་ཤོམ་ན་འདུག་པ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་ཉལ་པོ་བྱེད་པར་ཤོམ་པའི་ཁྱིམ་དུ་ཕྱིས་གནོན་བྱས་ཏེ། སྟན་ལ་འདུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.3 ཉལ་ཤོམ་ན་འགྲེང་བ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་ཉལ་པོ་བྱེད་པར་ཤོམ་པའི་ཁྱིམ་དུ་དབེན་པ་སྐྱབས་ཡོད་པར་འགྲེང་ན་ལྟུང་བྱེད་དོ། །
# f6.2.e.4 གཅེར་བུ་པ་ལ་སྦྱིན་པ།
ཡང་དགེ་སློང་གང་གཅེར་བུ་པ་འམ། གཅེར་བུ་མ་ཀུན་དུ་རྒྱུ་བ་འམ། ཀུན་ཏུ་རྒྱུ་མོ་ལ་རང་གི་ལག་ནས་བཅའ་བ་འམ་བཟའ་བ་བྱིན་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.5 དམག་ལ་ལྟ་བ།
ཡང་དགེ་སློང་གང་དམག་ཆས་པ་ལ་ལྟར་འགྲོ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.6 དམག་གི་ནང་ན་གནས་པ།
ཡང་དགེ་སློང་གང་དམག་ཆས་པ་ལ་ལྟ་བར་འགྲོ་བ་དེ་ལྟ་བུའི་རྐྱེན་ཞིག་བྱུང་ན། དགེ་སློང་དེས་དམག་དེའི་ནང་དུ་ཞག་གཉིས་ཚུན་ཆད་གནས་པར་བྱའོ། །དེ་ལས་ལྷག་པར་གནས་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.7 དཔུང་གི་ཡན་ལག་ཉམས་སུ་མྱོང་བར་འདོད་པ།
དགེ་སློང་ཞག་གཉིས་དམག་དེའི་ནང་ན་གནས་པའི་ཚེ་ན་ཡང་གལ་ཏེ་བཤམས་པ་དཀྲུག་ཏུ་འགྲོ་འམ། རྒྱལ་མཚན་གྱི་མཆོག་གམ། དཔུང་གི་མཆོག་གམ། གཡུལ་བཤམས་པའི་དམག་ལ་ལྟ་བ་ཉམས་སུ་མྱོང་བར་བྱེད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.8 བརྡེག་པ་ལ།
ཡང་དགེ་སློང་གང་ཁྲོས་འཁྲུགས་རྔམ་པར་གྱུར་ཏེ། ཡིད་མ་རངས་ནས་དགེ་སློང་ལ་བརྡེག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.5.9 བརྡེག་པར་གཟས་པ།
ཡང་དགེ་སློང་གང་ཁྲོས་འཁྲུགས་རྔམ་པར་གྱུར་ཏེ། ཡིད་མ་རངས་ནས་དགེ་སློང་ལ་བརྡེག་པར་གཟས་ན་ཐ་ན་ཐལ་མོས་ཀྱང་རུང་སྟེ་ལྟུང་བྱེད་དོ། །
# f6.2.5.10 གནས་ངན་ལེན་འཆབ་པ།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་དགེ་སློང་གི་གནས་ངན་ལེན་གྱི་ལྟུང་བ་འཆབ་ན་ལྟུང་བྱེད་དོ། །

# f6.2.6 བཅུ་ཚན་དྲུག་པ། ཁྱིམ་དང་གི་སྡེ།
# f6.2.6.a སྡོམ་ལ།
སྡོམ་ལ། བདེ་དང་མེ་དང་འདུན་པ་དང། །བསྙེན་པར་མ་རྫོགས་ཆོས་དང་སྨྲ། །དགེ་ཚུལ་ཁ་དོག་བསྒྱུར་བ་དང། །རིན་པོ་ཆེ་དང་ཚ་བའི་དུས། །
# f6.2.6.1 ཟན་གཅོད་འཇུག་པ།
ཡང་དགེ་སློང་གང་ཀླན་ཀ་ཚོལ་ཞིང་ཅི་ནས་ཀྱང་དགེ་སློང་འདི་ལ་ཀླན་ཀ་བཙལ་བར་འགྱུར་རོ་སྙམ་པ་དེ་ཉིད་རྐྱེན་དུ་བྱས་ཏེ། དགེ་སློང་ལ་འདི་སྐད་ཅེས་ཚེ་དང་ལྡན་པ་ཚུར་ཤོག་ཁྱིམ་དག་དུ་འདོང་དང་ཁྱོད་ལ་བཅའ་བ་དང་། བཟའ་བ་བསོད་པ་ཅི་ཙམ་འདོད་པ་སྦྱིན་དུ་གཞུག་གོ །ཞེས་སྨྲས་ནས་དེས་དེ་ལ་སྦྱིན་དུ་མ་བཅུག་པར་དེའི་འོག་ཏུ་འདི་སྐད་ཅེས་ཚེ་དང་ལྡན་པ་ཁོ་བོ་ལ་ཁྱོད་དང་ལྷན་ཅིག་སྨྲ་བ་འམ་འདུག་པ་བདེ་བ་མ་ཡིན་གྱི། འདི་ལྟར་ཁོ་བོ་གཅིག་པུ་ཁོ་ན་སྨྲ་འམ། འདུག་ན་བདེས་ཁྱོད་ཉིད་སོང་ཤིག་ཅེས་ཟེར་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.2 མེ་ལ་རེག་པ་ནི།
ཡང་དགེ་སློང་གང་མི་ན་བར་བདག་ཉིད་ཀྱི་ཕྱིར་མེ་ལ་རེག་གམ། རེག་དུ་འཇུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.3 འདུན་པ་ཕྱིར་བསྒྱུར་ནི།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་བྱ་བ་ཆོས་དང་ལྡན་པ་ལ་དགེ་སློང་ལ་འདུན་པ་ཕུལ་ནས། དེའི་འོག་ཏུ་ཁྲོས་འཁྲུགས་རྔམ་པར་གྱུར་ཏེ། ཡིད་མ་རངས་ནས་སྤོང་བའི་ཆོས་སུ་བྱེད་ཅིང་འདི་སྐད་ཅེས་དགེ་སློང་འདུན་པ་ཕྱིར་བྱིན་ཅིག་ཁྱོད་ལ་མི་སྦྱིན་ནོ་ཞེས་ཟེར་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.4 བསྙེན་པར་མ་རྫོགས་པ་དང་ལྷན་ཅིག་ཏུ་ཉལ་བ་ལ།
ཡང་དགེ་སློང་གང་གང་ཟག་བསྙེན་པར་མ་རྫོགས་པ་དང་ལྷན་གཅིག་ཏུ་ནུབ་གཉིས་ལས་ལྷག་པར་གནས་གཅིག་ཏུ་ཉལ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.5 སྡིག་ལྟ་མི་གཏོང་བ་ལ།
ཡང་དགེ་སློང་གང་འདི་སྐད་ཅེས་ཇི་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་བར་དུ་གཅོད་པའི་ཆོས་སུ་གསུངས་པ་གང་དག་ཡིན་པ་དེ་དག་བསྟེན་ཀྱང་བར་དུ་གཅོད་པར་མི་འགྱུར་ཏེ། དེ་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་བསྟན་པ་བདག་གིས་ཤེས་སོ། །ཞེས་ཟེར་ན། དགེ་སློང་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས་ཚེ་དང་ལྡན་པ་ཁྱོད་ཇི་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་བར་དུ་ གཅོད་པའི་ཆོས་སུ་གསུངས་པ་གང་དག་ཡིན་པ་དེ་དག་བསྟེན་ཀྱང་བར་དུ་གཅོད་པར་མི་འགྱུར་ཏེ། དེ་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་བསྟན་པ་བདག་གིས་ཤེས་སོ་ཞེས་དེ་སྐད་ཅེས་མ་ཟེར་ཅིག །བཅོམ་ལྡན་འདས་ལ་སྐུར་བ་མ་འདེབས་ཤིག ། བཅོམ་ལྡན་འདས་ལ་བསྐུར་བས་ལེགས་པར་མི་འགྱུར་རོ། །བཅོམ་ལྡན་འདས་ནི་དེ་སྐད་མི་གསུང་ངོ། །ཚེ་དང་ལྡན་པ་བར་དུ་གཅོད་པའི་ཆོས་རྣམས་ནི་བར་དུ་གཅོད་པ་ཉིད་དོ་ཞེས་བཅོམ་ལྡན་འདས་ཀྱིས་རྣམ་གྲངས་དུ་མར་གསུངས་ཏེ། དེ་དག་བསྟེན་ན་བར་དུ་གཅོད་པར་འགྱུར་གྱིས་ཚེ་དང་ལྡན་པ་ཁྱོད་སྡིག་པ་ཅན་གྱི་ལྟ་བའི་རྣམ་པ་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་སློང་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་བསྒོ་བ་ན་གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་ གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱ། ཡང་དག་པར་བསྟན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། མི་གཏོང་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.6 སྤངས་པ་རྗེས་ཕྱོགས་ནི།
ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་དེ་སྐད་ཟེར་བའི་གང་ཟག་ཆོས་བཞིན་དུ་མ་བྱས་པ་སྡིག་པ་ཅན་གྱི་ལྟ་བའི་རྣམ་པ་དེ་མ་སྤངས་པ་དང་གཏམ་འདྲེ་བར་བྱེད། ཕེབས་པར་སྨྲ་བར་བྱེད། ཀུན་ཏུ་གནས་པར་བྱེད། ཀུན་ཏུ་ལོངས་སྤྱོད་པར་བྱེད་ཅིང། དེ་དང་ལྷན་ཅིག་གནས་གཅིག་ཏུ་ཉལ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.7 བསྙིལ་བ་སྡུད་པ།
དགེ་ཚུལ་ཞིག་ཀྱང་འདི་སྐད་ཅེས་ཇི་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་འདོད་པ་དག་ནི་བར་དུ་གཅོད་པའོ་ཞེས་གསུངས་པ་གང་དག་ཡིན་པ་དེ་དག་བསྟེན་ཀྱང་བར་དུ་གཅོད་པར་མི་འགྱུར་ཏེ། དེ་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་བསྟན་པ་བདག་གིས་ཤེས་སོ་ཞེས་ཟེར་ན། དགེ་ཚུལ་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་འདི་སྐད་ཅེས་དགེ་ཚུལ་ཁྱོད་ཇི་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་འདོད་པ་དག་ནི་བར་དུ་གཅོད་པ་འོ་ཞེས་གསུངས་པ་གང་ཡིན་པ་དེ་དག་བསྟེན་ཀྱང་བར་དུ་གཅོད་པར་མི་འགྱུར་ཏེ། དེ་ལྟར་བཅོམ་ལྡན་འདས་ཀྱིས་ཆོས་བསྟན་པ་བདག་གིས་ཤེས་སོ་ཞེས་དེ་སྐད་མ་ཟེར་ཅིག །བཅོམ་ལྡན་འདས་ལ་སྐུར་པ་མ་འདེབས་ཤིག །བཅོམ་ལྡན་འདས་ལ་སྐུར་པས་ལེགས་པར་མི་འགྱུར་རོ། །བཅོམ་ལྡན་འདས་ནི་དེ་སྐད་མི་གསུང་ངོ། །ཚེ་དང་ལྡན་པ་དགེ་འཚུལ་འདོད་པ་བར་དུ་གཅོད་པ་རྣམས་ནི་བར་དུ་གཅོད་པ་ཉིད་དོ་ཞེས་བཅོམ་ལྡན་འདས་ཀྱིས་རྣམ་གྲངས་དུ་མར་ གསུངས་ཏེ། དེ་དག་བསྟེན་ན་བར་དུ་གཅོད་པར་འགྱུར་གྱིས། དགེ་ཚུལ་ཁྱོད་སྡིག་པ་ཅན་གྱི་ལྟ་བའི་རྣམ་པ་འདི་ལྟ་བུ་འདི་ཐོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །དགེ་ཚུལ་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེ་སྐད་བསྒོ་བ་ན་གལ་ཏེ་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་གཞི་དེ་གཏོང་བར་བྱ་བའི་ཕྱིར་ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ་བར་བྱ། ཡང་དག་པར་བསྟན་པར་བྱའོ། །ལན་གཉིས་ལན་གསུམ་དུ་ཡང་དག་པར་བསྒོ། ཡང་དག་པར་བསྟན་པ་ན་གཞི་དེ་གཏོང་ན་དེ་ལྟ་ན་ལེགས། གལ་ཏེ་མི་གཏོང་ན་དགེ་ཚུལ་དེ་ལ་དགེ་སློང་རྣམས་ཀྱིས་དེང་ཕྱིན་ཆད་དགེ་ཚུལ་ཁྱོད་བཅོམ་ལྡན་འདས་དེ་བཞིན་གཤེགས་པ་དགྲ་བཅོམ་པ་ཡང་དག་པར་རྫོགས་པའི་སངས་རྒྱས་དེ་ལ་སྟོན་པའོ་ཞེས་མ་ཟེར་ཅིག །ཚངས་པ་མཚུངས་པར་སྤྱོད་པ་མཁས་པ་བླ་མའི་གནས་ལྟ་བུ་གང་ཡང་རུང་བའི་ཕྱི་བཞིན་དུ་ཡང་མ་འགྲོ་ཤིག །དགེ་སློང་རྣམས་དང་དགེ་ཚུལ་ལྷན་ཅིག་ནུབ་གཉིས་ཚུན་ཆད་གནས་གཅིག་ཏུ་ཉལ་དུ་དབང་བ་གང་ཡིན་པ་དེ་ཡང་དེང་ཕྱིན་ཆད་ཁྱོད་ལ་མེད་དེ། མི་གཏི་མུག་ཅན་ཁྱོད་བསྙིལ་གྱིས་གཞན་དུ་སོང་ཤིག་ཅེས་བསྒོ་བར་བྱའོ། །ཡང་དགེ་སློང་གང་ཤེས་བཞིན་དུ་དེ་ལྟར་བསྙིལ་བའི་དགེ་ཚུལ་ཉེ་བར་འཇོག་པར་བྱེད་དམ། ཉེ་བར་སྟོན་པར་བྱེད་དམ། དེ་དང་ལྷན་ཅིག་གནས་གཅིག་ཏུ་ཉལ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.8 ཁ་མ་བསྒྱུར་བ་གྱོན་པ་ལ།
དགེ་སློང་གིས་གོས་སར་པ་ཞིག་རྙེད་ན་ཁ་བསྒྱུར་བ་གསུམ་པོ་སྔོན་པོ་འམ། དམར་པོ་འམ། ངུར་སྨྲིག་ལས་གང་ཡང་རུང་བས་ཁ་དོག་བསྒྱུར་བར་བྱའོ། །གལ་ཏེ་དགེ་སློང་གིས་གོས་སར་པ་ཁ་བསྒྱུར་བ་གསུམ་པོ་སྔོན་པོ་འམ། དམར་པོ་འམ། ངུར་སྨྲིག་ལས་གང་ཡང་རུང་བར་ཁ་མ་བསྒྱུར་བར་ལོངས་སྤྱོད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.6.9 རིན་པོ་ཆེའི་གདགས་གཞིར་ལོངས་པ་ལེན་པ།
ཡང་དགེ་སློང་གང་རིན་པོ་ཆེ་འམ། རིན་པོ་ཆེར་སྨོས་པ་རང་གི་ལག་གིས་ལེན་ཏམ། ལེན་དུ་འཇུག་ན་ཀུན་དགའ་ར་བར་གཏོགས་པ་ན་འདུག་པ་འམ། གནས་ཁང་དུ་གཏོགས་པ་ན་འདུག་པ་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.6.10 ཁྲུས་བྱེད་པ་ལ།
དགེ་སློང་གིས་ཀུན་དགའ་ར་བར་གཏོགས་པ་འམ་གནས་ཁང་དུ་གཏོགས་པ་ན་རིན་པོ་ཆེ་འམ། རིན་པོ་ཆེར་སྨོས་པ་འདུག་པ་ལ་འདི་སུའི་ཡིན་པ་དེ་ལེན་པར་འགྱུར་གྲང་སྙམ་པ་དེ་ལྟ་བུའི་སེམས་ཀྱིས་བླང་བར་བྱ་སྟེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །བཅོམ་ལྡན་འདས་ཀྱིས་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་ཁྲུས་བྱ་བར་གསུངས་པ་དེ་ལས་འདུམས་ པར་བྱེད་ན་དུས་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །དེ་ལ་དུས་ནི་འདི་ཡིན་ཏེ། སོ་ཀ་རྣམས་ཀྱི་ཟླ་བ་ལྷག་མ་ཕྱེད་དང་གཉིས་དང། དབྱར་རྣམས་ཀྱི་དང་པོ། ཟླ་བ་ཕྱེད་དང་གསུམ་པོ་འདི་དག་ནི་ཚ་བའི་དུས་སོ། །ལྷག་མ་ནི་ན་བ་དུས་སོ། །ལས་དུས་སོ། །རླུང་དུས་སོ། །ཆར་དུས་སོ། །རྫི་ཆར་དུས་ཏེ། དེ་ལ་དེ་ནི་དུས་ཡིན་ནོ། །
# f6.2.7 བཅུ་ཚན་བདུན་པ། བསམ་བཞིན་གྱི་སྡེ།
# f6.2.7.a སྡོམ་ལ།
སྡོམ་ལ། དུད་འགྲོ་འགྱོད་པ་སོར་མོ་དང། །རྩེ་དང་ལྷན་ཅིག་སྔངས་བྱེད་དང། །སྦེད་དང་གདེང་མེད་གཞི་མེད་དང། ། སྐྱེས་པ་མེད་པར་ལམ་འགྲོ་བའོ། །
# f6.2.7.1 དེ་ལ་དུད་འགྲོ་གསོད་པ་ནི།
ཡང་དགེ་སློང་གང་བསམས་བཞིན་དུ་དུད་འགྲོའི་སྐྱེ་གནས་སུ་གཏོགས་པའི་སྲོག་ཆགས་གསོད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.2 འགྱོད་པ་བསྐྱེད་པ་ནི།
ཡང་དགེ་སློང་གང་བསམས་བཞིན་དུ་ཅི་ནས་ཀྱང་དགེ་སློང་འདི་ཡུད་ཙམ་ཞིག་ཀྱང་བདེ་བ་ལ་མི་རེག་པར་བྱའོ་སྙམ་པ་དེ་ཉིད་རྐྱེན་དུ་བྱས་ཏེ། དགེ་སློང་ལ་འགྱོད་པ་སྐྱེད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.3 གག་ཚིལ་ནི།
སོར་མོས་ག་ག་ཚིལ་བྱེད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.4 ཆུ་ལ་རྩེ་བ་ནི།
ཆུ་ལ་རྩེ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.5 བུད་མེད་དང་ལྷན་ཅིག་ཉལ་བ་ནི།
ཡང་དགེ་སློང་གང་བུད་མེད་ཀྱི་ཡུལ་དང་ལྷན་ཅིག་གནས་སུ་ཉལ་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.6 སྔངས་པར་བྱེད་པ་ལ།
ཡང་དགེ་སློང་གང་དགེ་སློང་ལ་སྔངས་པར་བྱེད་དམ། སྔངས་པར་བྱེད་དུ་འཇུག་ན་ཐ་ན་བཞད་གད་བྱ་བར་བསམས་ཀྱང་རུང་སྟེ་ལྟུང་བྱེད་དོ། །
# f6.2.7.7 སྦེད་པ་ལ།
ཡང་དགེ་སློང་གང་དགེ་སློང་ངམ། དགེ་སློང་མ་འམ། དགེ་སློབ་མ་འམ། དགེ་ཚུལ་ལམ། དགེ་ཚུལ་མའི་ལྷུང་བཟེད་དམ། ཆོས་གོས་སམ། དྲ་བ་འམ། ཕོར་བུ་འམ། སྐ་རགས་སམ། དགེ་སྦྱོང་གི་འཚོ་བའི་ཡོ་བྱད་གང་ཡང་རུང་བ་སྦེད་དམ་སྦེད་དུ་འཇུག་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.7.8 གདིང་མེད་དུ་སྤྱོད་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་ལ་གོས་བྱིན་ནས་དེའི་འོག་ཏུ་གདེང་མེད་པར་སྤྱོད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.9 ལྷག་མས་སྐུར་འདེབས་ནི།
ཡང་དགེ་སློང་གང་ཁྲོས་ཤིང་ཞེ་སྡང་བར་གྱུར་ནས་དགེ་སློང་དག་པ་ལྟུང་བ་མེད་པ་ལ་གཞི་མེད་པར་དགེ་འདུན་ལྷག་མའི་ཆོས་ཀྱིས་སྐུར་ན་ལྟུང་བྱེད་དོ། །
# f6.2.7.10 བུད་མེད་དང་ལྷན་ཅིག་ལམ་འགྲོ་ནི།
ཡང་དགེ་སློང་གང་སྐྱེས་པ་མེད་པར་བུད་མེད་དང་ལྷན་ཅིག་འདྲོན་ལམ་དུ་འགྲོ་ན་ཐ་ན་གྲོང་བར་དུ་ཡང་རུང་སྟེ་ལྟུང་བྱེད་དོ། །

# f6.2.8 བཅུ་ཚན་བརྒྱད་པ། མགྲོན་མང་གི་སྡེ།
# f6.2.8.a སྡོམ་ལ།
སྡོམ་ལ། རྐུ་དང་ཉི་ཤུ་མ་ལོན་དང། །རྐོ་དང་མགྲོན་དང་བསླབ་པ་དང། །འཐབ་དང་མི་སྨྲ་འགྲོ་བ་དང། །མི་གུས་ཆང་ འཐུང་དུས་མིན་པའོ། །
# f6.2.8.1 རྐུན་མ་དང་ལྷན་ཅིག་ལམ་འགྲོ་ནི།
ཡང་དགེ་སློང་གང་དོན་འཐུན་པ་རྐུན་མ་དང་ལྷན་ཅིག་འགྲོན་ལམ་དུ་འགྲོ་ན་ཐ་ན་གྲོང་བར་དུ་ཡང་རུང་སྟེ་ལྟུང་བྱེད་དོ། །
# f6.2.8.2 ཉི་ཤུ་མ་ལོན་པར་རྫོགས་པ་ནི།
ཡང་དགེ་སློང་གང་གང་ཟག་ལོ་ཉི་ཤུ་མ་ལོན་པ་དགེ་སློང་གི་དངོས་པོར་བསྙེན་པར་རྫོགས་པར་བྱེད་ན་ལྟུང་བྱེད་དོ། །གང་ཟག་དེ་ཡང་བསྙེན་པར་རྫོགས་པར་མི་འགྱུར་ལ། དགེ་སློང་དེ་དག་ཀྱང་སྨད་པར་འགྱུར་བ་ཡིན་ཏེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f6.2.8.3 ས་བརྐོ་བ་ནི།
ཡང་དགེ་སློང་གང་རང་གི་ལག་གིས་ས་རྐོ་འམ་རྐོར་འཇུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.8.4 མགྲོན་དུ་བོས་པ་ལས་རིང་དུ་བསྡད་པ་ནི།
དགེ་སློང་གིས་ཟླ་བ་བཞིར་འགྲོན་དུ་བོས་པ་བདག་གིར་བྱའོ། །དེ་ལས་ལྷག་པར་བདག་གིར་བྱེད་ན་ལྟུང་བྱེད་དོ། །སོ་སོར་མགྲོན་དུ་བོས་པ་དང། ཡང་དང་ཡང་དུ་མགྲོན་དུ་བོས་པ་དང། དུས་ཀྱི་མགྲོན་དུ་བོས་པ་དང། རྟག་ཏུ་མགྲོན་དུ་བོས་པ་ནི་མ་གཏོགས་ཏེ་དེ་ལ་དེ་ནི་དུས་ཡིན་ནོ། །
# f6.2.8.5 བསླབ་པ་ཉེར་འཇོག་སྤོང་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་རྣམས་ཀྱིས་ཚེ་དང་ལྡན་པ་ཁྱོད་ཀྱིས་བསླབ་པ་འདི་ལ་བསླབ་པར་བྱའོ་ཞེས་བསྒོ་བ་ན་དེ་དག་ལ་འདི་སྐད་ཅེས་བདག་དགེ་སློང་མདོ་སྡེ་འཛིན་པ་དང་། འདུལ་བ་འཛིན་པ་དང། མ་མོ་འཛིན་པ་རྣམས་ལ་འདྲི་བའི་བར་དུ་བྱིས་པ་རྨོངས་པ་མི་གསལ་བ་མི་མཁས་པ་ཁྱེད་ཀྱི་ཚིག་གིས་བསླབ་པ་འདི་ལ་མི་སློབ་བོ་ཞེས་ཟེར་ན་ལྟུང་བྱེད་དོ། །དགེ་སློང་ཀུན་ཤེས་པར་འདོད་པས་ཀྱང་བསླབ་པ་དེ་ལ་བསླབ་པར་བྱའོ། །དགེ་སློང་མདོ་སྡེ་འཛིན་པ་དང། འདུལ་བ་འཛིན་པ་དང། མ་མོ་འཛིན་པ་རྣམས་ལ་ཡང་དྲི་བར་བྱ་སྟེ། དེ་ལ་དེ་ནི་ཆོ་ག་ཡིན་ནོ། །
# f6.2.8.6 ཉན་རྣ་ལས་གྱུར་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་རྣམས་འཐབ་པར་གྱུར་མཚང་འདྲུ་བར་གྱུར། མི་མཐུན་པར་གྱུར། རྩོད་པར་གྱུར་ཅིང་འཁོད་པ་ལས། དགེ་སློང་འདི་དག་ཇི་སྐད་ཟེར་བ་དེ་བདག་གིས་མཉན་ཏེ། དེ་དེ་ལྟར་རྗེས་སུ་ཐ་སྙད་གདགས་པར་བྱའོ་སྙམ་པ་དེ་ཉིད་རྐྱེན་དུ་བྱས་ཏེ། ཅང་མི་སྨྲ་བར་ཉན་རྣས་འདུག་ན་ལྟུང་བྱེད་དོ། །
# f6.2.8.7 མི་སྨྲ་བར་འགྲོ་བ་ནི།
ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་ཆོས་དང་ལྡན་པར་གཏམ་རྣམ་པར་གཏན་ལ་འབེབས་པར་བྱུང་བ་ན། ཅང་མི་སྨྲ་བར་སྟན་ལས་ལངས་ཏེ་འགྲོ་ཞིང་དགེ་སློང་འདུག་པ་ལ་མི་སྨྲ་ན། དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །
# f6.2.8.8 མ་གུས་པ་ལ།
གུས་པར་མི་བྱེད་ན་ལྟུང་བྱེད་དོ། །
# f6.2.8.9 མྱོས་འགྱུར་འཐུང་བ་ནི།
འབྲུའི་ཆང་དང་བཅོས་པའི་ཆང་མྱོས་པར་འགྱུར་བ་འཐུང་ན་ལྟུང་བྱེད་དོ། །
# f6.2.8.10 དུས་མིན་པར་འགྲོ་བ་ལ།
ཡང་དགེ་སློང་གང་དུས་མ་ཡིན་པར་གྲོང་དུ་འཇུག་ཅིང་དགེ་སློང་འདུག་པ་ལ་མི་སྨྲ་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །

# f6.2.9 བཅུ་ཚན་དགུ་པ། ཆོས་སྟོན་གྱི་སྡེ།

# f6.2.9.2 རྒྱལ་པོའི་ཕོ་བྲང་དུ་ནུབ་མོ་འགྲོ་བ་ནི།
ཡང་དགེ་སློང་གང་ནམ་མ་ནངས་སྐྱ་རེང་མ་ཤར་བར་རིན་པོ་ཆེ་དག་གམ་རིན་པོ་ཆེར་སྨོས་པ་དག་མ་བསྡུས་པར་རྒྱལ་པོ་རྒྱལ་རིགས་སྤྱི་བོར་དབང་བསྐུར་བའི་སྒོ་གཏན་ནམ། སྒོ་གཏན་གྱི་ཉེ་འཁོར་ལས་འདས་ན་དེ་འདྲ་བའི་རྐྱེན་མ་གཏོགས་ཏེ་ལྟུང་བྱེད་དོ། །

# f7.a སོ་སོར་བཤགས་པ།
# f7.a སོ་སོར་བཤགས་པ་ལ། སྒྲོ་སྐུར་སྤངས་དོན་དུ་སྡོམ་ནི།
སྡོམ་ལ། གྲོང་དང་ཁྱིམ་གཞན་ཉིད་དང་ནི། །སླབ་པ་རྣམས་དང་དགོན་པ་ནི། །སངས་རྒྱས་ཕན་པ་གསུངས་པ་ཡིས། །སོ་སོར་བཤགས་པར་བྱ་བར་གསུངས། །
# f7.b འདོན་པའི་ལུང་འབོག་པ་ནི།
ཚེ་དང་ལྡན་པ་དག་སོ་སོར་བཤགས་པར་བྱ་བའི་ཆོས་བཞི་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# f7.c.1 དགེ་སློང་མ་ལས་ཟ་མ་ལེན་པའི་སོ་སོར་བཤགས་པ་ནི།
ཡང་དགེ་སློང་གང་དགེ་སློང་མ་ཉེ་དུ་མ་ཡིན་པ་ལམ་པོ་ཆེ་ན་འདུག་པ་བསོད་སྙོམས་ཀྱི་ཕྱིར་གྲོང་དུ་འགྲོ་བ་ལས་བཅའ་བ་དང་བཟའ་བ་རང་གི་ལག་གིས་བསླངས་ཏེ་འཆའ་བ་འམ། ཟ་ན་དགེ་སློང་དེས་ཕྱི་རོལ་ཀུན་དགའ་ར་བར་སོང་སྟེ། དགེ་སློང་རྣམས་ལ་ཚེ་དང་ལྡན་པ་དག་བདག་ལ་སྨད་པའི་གནས་མི་རིགས་པ་སོ་སོར་བཤགས་པར་བྱ་བ་བྱུང་གིས་ཆོས་དེ་སོ་སོར་འཆགས་སོ་ཞེས་སོ་སོར་བཤགས་པར་བྱ་བ་སྟེ། ཆོས་འདི་ནི་སོ་སོར་བཤགས་པར་བྱ་བའོ། །
# f7.c.2 གྲལ་རིམ་དུ་མ་ཕེབ་པར་འབྲིམ་དུ་འཇུག་པ་མ་བཟློག་པར་ཟ་བ་ནི།
དགེ་སློང་རབ་ཏུ་མང་པོ་དག་ཁྱིམ་རྣམས་སུ་མགྲོན་དུ་བོས་ནས་ཟན་ཟ་བའི་ཚེ་ན། གལ་ཏེ་དགེ་སློང་མ་ཞིག་འདི་ལ་ནི་བཅའ་བ་བྱིན་ཅིག །འདི་ལ་འབྲས་ཆན་བྱིན་ཅིག །འདི་ལ་ནི་སྲན་ཚོད་བྱིན་ཅིག །འདི་ལ་ནི་བཏུང་བ་བྱིན་ཅིག་ཅེས་བསྒོ་ཞིང་འདུག་ན། དགེ་སློང་མ་དེ་ལ་དགེ་སློང་དེ་རྣམས་ཀྱིས་འདི་སྐད་ཅེས་སྲིང་མོ་ཁྱོད་དགེ་སློང་རྣམས་ཟན་ཟ་བའི་བར་དུ་རེ་ཤིག་སྡོད་ཅིག་ཅེས་བསྒོ་བར་བྱའོ། །གལ་ཏེ་དགེ་སློང་གཅིག་གིས་ཀྱང་དགེ་སློང་མ་དེ་ལ་དེ་སྐད་སྨྲ་བ་མ་སྤོབས་ན། དགེ་སློང་དེ་དག་ཐམས་ཅད་ཀྱིས་ཕྱི་རོལ་ཀུན་དགའ་ར་བར་སོང་སྟེ། དགེ་སློང་རྣམས་ལ་ཚེ་དང་ལྡན་པ་དག་བདག་ཅག་ལ་སྨད་པའི་གནས་མི་རིགས་པ་སོ་སོར་བཤགས་པར་བྱ་བ་བྱུང་གིས་ཆོས་དེ་སོ་སོར་བཤགས་སོ་ཞེས་སོ་སོར་བཤགས་པར་བྱ་སྟེ། ཆོས་འདི་ཡང་སོ་སོར་བཤགས་པར་བྱ་བའོ། །
# f7.c.3 བསླབ་པ་དྲལ་ཏེ་ཁྱིམ་དུ་འཇུག་པ་ནི།
སློབ་པ་དག་གིས་ཁྱིམ་གང་དགེ་འདུན་གྱི་བསླབ་པའི་སྡོམ་པས་བསྡམས་པར་གྱུར་ལ། ཡང་དགེ་སློང་གང་སློབ་པ་དག་གིས་ཁྱིམ་གང་དགེ་འདུན་གྱི་བསླབ་པའི་སྡོམ་པས་བསྡམས་པ་དེ་ལྟ་བུ་དག་ཏུ་སྔར་མགྲོན་དུ་མ་བོས་པར་སོང་སྟེ། རང་གི་ལག་གིས་བཅའ་བ་དང་བཟའ་བ་བླངས་ནས་འཆའ་འམ་ཟ་ན། དགེ་སློང་དེས་ཕྱི་རོལ་ཀུན་དགའ་ར་བར་སོང་སྟེ། དགེ་སློང་རྣམས་ལ་ཚེ་དང་ལྡན་པ་དག་བདག་ལ་སྨད་པའི་གནས་མི་རིགས་པ་སོ་སོར་བཤགས་པར་བྱ་བ་བྱུང་གིས་ཆོས་དེ་སོ་སོར་བཤགས་སོ་ཞེས་སོ་སོར་བཤགས་པར་བྱ་སྟེ། ཆོས་འདི་ཡང་སོ་སོར་བཤགས་པར་བྱའོ། །
# f7.c.4 ནགས་མ་བརྟགས་པ་ལས་གྱུར་པ་ནི།
དགེ་འདུན་གྱི་གནས་མལ་དགོན་པ་གང་དག་དོགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་ཐ་དད་པས་འཇིགས་སུ་རུང་བ་དང་བཅས་པར་གྲགས་པར་འགྱུར་ལ། ཡང་དགེ་སློང་གང་དགེ་འདུན་གྱི་གནས་མལ་དགོན་པ་གང་དག་དོགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་དང་བཅས་པར་གྲགས་པ། འཇིགས་པ་ཐ་དད་པས་འཇིགས་སུ་རུང་བ་དང་བཅས་པར་གྲགས་པ་དེ་ལྟ་བུ་དག་ཏུ་སྔར་མ་ཉུལ་བར་ཀུན་དགའ་ར་བར་ཕྱི་རོལ་ཏུ་བཅའ་བ་དང་བཟའ་བ་བླངས་ཏེ་འཆའ་འམ་ཟ་ན། དགེ་སློང་དེས་ཕྱི་རོལ་ཀུན་དགའ་ར་བར་སོང་སྟེ། དགེ་སློང་རྣམས་ལ་ཚེ་དང་ལྡན་པ་དག་བདག་ལ་སྨད་པའི་གནས་མི་རིགས་པ་སོ་སོར་བཤགས་པར་བྱ་བ་བྱུང་གིས་ཆོས་དེ་སོ་སོར་བཤགས་སོ་ཞེས་སོ་སོར་བཤགས་པར་བྱ་སྟེ། ཆོས་འདི་ཡང་སོ་སོར་བཤགས་པར་བྱ་བའོ། །
# f7.d འདོན་པ་བསྡུ་བ་ནི།
ཚེ་དང་ལྡན་པ་དག་བདག་གིས་སོ་སོར་བཤགས་པར་བྱ་བའི་ཆོས་བཞི་པོ་དག་བཏོན་ཟིན་ཏོ། །
# f7.e རྣམ་གཞག་བསྒྲགས་པ་ནི།
དེ་ལ་བདག་གིས་ཚེ་དང་ལྡན་པ་དག་ལ་ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་དྲིའོ། །ཅི་འདི་ལ་ཁྱེད་ཡོངས་སུ་དག་གམ་ཞེས་ལན་གཉིས་ལན་གསུམ་གྱི་བར་དུ་དྲིའོ། །འདི་ལ་ཚེ་དང་ལྡན་པ་དག་ཡོངས་སུ་དག་ན་འདི་ལྟར་ཅང་མི་སྨྲ་བས་དེ་དེ་བཞིན་དུ་འཛིན་ཏོ། །

# f8  ཉེས་བྱས་ཀྱི་སྡེ།
# f8.a ཉེས་བྱས་ཀྱི་སྡེ་ལས་སྒྲོ་སྐུར་སྤང་པ་ནི།
སྡོམ་ལ། ཤམ་ཐབས་ལ་ནི་རྣམ་བདུན་དང། །སྟོད་གཡོགས་ལ་ཡང་རྣམ་གསུམ་དང། །ཤིན་ཏུ་བསྡམས་ལ་སོགས་པ་ལྔ། །མགོ་གཡོགས་ལ་སོགས་རྣམ་པ་ལྔ། །མཆོང་ལ་སོགས་པ་རྣམ་ལྔ་དང། །ལུས་ལ་སོགས་པ་རྣམ་པ་ལྔ། །འདུག་པར་བྱ་བ་དགུ་དག་དང། །བྱིན་ལེན་བྱ་བ་བརྒྱད་རྣམས་སོ། །
# f8.b འདོན་པའི་ཆེད་དུ་ལུང་འབོགས་པ་ནི།
ཚེ་དང་ལྡན་པ་དག་བསླབ་པའི་ཆོས་མང་པོ་འདི་དག་རྣམས་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# f8.c.1 དེ་ལ་གོས་བསྒོ་བའི་སྡེ་ནི།
ཤམ་ཐབས་ཟླུམ་པོར་བགོ་བར་བསླབ་པར་བྱ། ཤམ་ཐབས་ཧ་ཅང་རྩེངས་པ་མ་ཡིན་པ་དང། ཧ་ཅང་འཇོལ་བ་མ་ཡིན་པ་དང། གླང་པོ་ཆེའི་སྣ་ལྟར་མ་ཡིན་པ་དང། ཏ་ལའི་ལོ་མ་ལྟར་བལྟབ་པ་མ་ཡིན་པ་དང། འབྲུའི་ཕུར་མ་ལྟར་མ་ཡིན་པ་དང། སྦྲུལ་མགོའི་གདེངས་ཀ་ལྟར་མ་ཡིན་པར་བགོ་བར་བསླབ་པར་བྱའོ། །ཆོས་གོས་ཟླུམ་པོར་བགོ་བར་བསླབ་པར་བྱའོ། །ཆོས་གོས་ཧ་ཅང་རྩེངས་པ་མ་ཡིན་པ་དང་ཧ་ཅང་འཇོལ་བ་མ་ཡིན་པར་བགོ་བར་བསླབ་པར་བྱའོ། །
# f8.c.2 ལམ་འཇུག་གི་སྡེ་ལ།
ཤིན་ཏུ་བསྡམས་པ་དང། ལེགས་པར་བགོས་པ་དང། སྒྲ་སྐྱུང་བ་དང། མིག་གཡེང་བར་མི་བྱ་བ་དང། གཉའ་ཤིང་གང་ཙམ་དུ་བལྟ་ཞིང་ཁྱིམ་གཞན་དག་ཏུ་འགྲོ་བར་བསླབ་པར་བྱའོ། །མགོ་མི་གཡོགས་པ་དང་མི་བརྫེ་བ་དང་མི་གཟར་བ་དང། གཉའ་གོང་དུ་མི་བསྣོལ་བ་དང་ལྟག་པར་མི་བསྣོལ་བར་ཁྱིམ་གཞན་དུ་འགྲོ་བར་བསླབ་པར་བྱའོ། །མི་མཆོང་བ་དང་མི་བརྐྱང་བ་དང། ཙོག་པུས་མ་ཡིན་པ་དང། བྲང་བས་མ་ཡིན་པ་དང། དཀུར་མི་བརྟེན་པར་ཁྱིམ་གཞན་དུ་འགྲོ་བར་བསླབ་པར་བྱའོ། །ལུས་མི་བསྒྱུར་བ་དང། ལག་པ་མི་དཀྱོག་པ་དང། མགོ་མི་བསྒྱུར་བ་དང། ཕྲག་པ་མི་སྤྲད་པ་དང། ལག་མི་སྦྲེལ་བར་ཁྱིམ་གཞན་དུ་འགྲོ་བར་བསླབ་པར་བྱའོ། །
# f8.c.3 སྟན་ལ་འདུག་པའི་སྡེ་ལ།
མ་བསྒོ་བར་ཁྱིམ་གཞན་དུ་སྟན་ལ་མི་འདུག་པར་བསླབ་པར་བྱ། སྟན་ལ་མ་བརྟགས་པ་དང་ལུས་ཐམས་ཅད་ཀྱི་ལྗིད་ཀྱིས་མི་དབབ་པ་དང། རྐང་པ་མི་བསྣོལ་བ་དང། བརླ་མི་བསྣོལ་བ་དང། ལོང་བུའི་སྟེང་དུ་ལོང་བུ་མི་གཞག་པ་དང། རྐང་པ་མི་དགུག་པ་དང། རྐང་པ་མི་གདང་བ་དང། འདོམས་མི་སྣང་བར་ཁྱིམ་གཞན་དུ་སྟན་ལ་འདུག་པར་བསླབ་པར་བྱའོ། །
# f8.c.4 ཟས་བསླང་བའི་སྡེ་ནི།
ལེགས་པར་ཟས་བླང་བར་བསླབ་པར་བྱ། མུ་དང་ཁ་ད་ཆད་དུ་མ་ཡིན་པ་དང། ཚོད་མ་དང་མཉམ་པར་མ་ ཡིན་པ་དང། ཐར་ཆགས་སུ་དང་ལྷུང་བཟེད་ལ་བལྟ་བ་དང། བཅའ་བ་དང་བཟའ་བ་མ་འོངས་པར་ལྷུང་བཟེད་མི་བཟེད་པ་དང་ཡང་འདོད་པའི་ཕྱིར་འབྲས་ཅན་གྱིས་ཚོད་མ་མི་དགབ་ཚོད་མས་འབྲས་ཆན་མི་དགབ་པ་དང། བཅའ་བ་དང་ བཟའ་བའི་སྟེང་དུ་ལྷུང་བཟེད་མི་གཟུང་བར་བསླབ་པར་བྱའོ། །
# f8.c.5 ཟས་བཟའ་བའི་སྡེ།
# f8.c.5.a སྡོམ་ནི། 
སྡོམ་ལ། ཟས་ལ་ལེགས་པར་བྱ་བ་དྲུག །ཙུག་ཙུག་ལ་སོགས་རྣམ་པ་ལྔ། །འབྲུ་ནས་ཐ་དད་བྱེད་པ་ལྔ། །ལག་པ་བལྡག་ལ་སོགས་པ་ལྔ། །
# f8.c.5.b ཟས་བཟའ་བའི་སྡེ་ལ།
ལེགས་པར་ཟས་བཟའ་བར་བསླབ་པར་བྱ། ཁམ་ཧ་ཅང་ཆུང་བ་མ་ཡིན་པ་དང། ཁམ་ཧ་ཅང་ཆེ་བ་མ་ཡིན་པ་དང་། ཁམ་རན་པ་དང་། ཁམ་མ་གཟས་པར་ཁ་མི་གདང་བ་དང། ཁ་ཁམ་གྱིས་བཀང་སྟེ་མི་སྨྲ་བར་བསླབ་པར་བྱའོ། །ཙུག་ཙུག་མི་བྱ་བ་དང། བལྕག་བལྕག་མི་བྱ་བ་དང་། ཧུ་ཧུ་མི་བྱ་བ་དང་། ཕུ་ཕུ་མི་བྱ་བ་དང་། ལྕེ་ཕྱུང་སྟེ་ཟས་མི་བཟའ་བར་བསླབ་པར་བྱའོ། །འབྲུ་ནས་ཐ་དད་དུ་མི་བྱ་བ་དང་། འཕྱས་མི་གདགས་པ་དང་མཁུར་བ་མི་སྤོ་བ་དང། དཀན་མི་གཏོགས་པ་དང་། ཁམ་འཕྲོར་མི་བཅད་པར་ཟས་བཟའ་བར་བསླབ་པར་བྱའོ། །ལག་པ་མི་བལྡག་པ་དང་ལྷུང་བཟེད་མི་བྱོག་པ་དང། ལག་པ་མི་སྤྲུག་པ་དང་ལྷུང་བཟེད་མི་བསྐྱོམ་པ་དང། མཆོད་རྟེན་འདྲ་བར་བཅོས་ཏེ་ཟས་མི་བཟའ་བར་བསླབ་པར་བྱའོ། །

# f8.c.6 ལྷུང་བཟེད་སྤྱད་པ་ལ་སོགས་པའི་སྡེ་རྣམས།
# f8.c.6.a ལྷུང་བཟེད་སྤྱད་པ་ལ་སོགས་པའི་སྡེ་རྣམས་ལ། སྡོམ་ནི།
སྡོམ་ལ། འཕྱ་ལ་སོགས་པ་རྣམ་པ་བཞི། །ལྷུང་བཟེད་ལ་ཡང་རྣམ་པ་བཅུ། །འགྲེང་བར་བྱེད་ལ་སོགས་པ་ལྔ། །མགོ་གཡོགས་ལ་སོགས་རྣམ་པ་ལྔ། །དོ་ཀེར་ཅན་ལ་སོགས་པ་ལྔ། །གླང་ཆེན་ལ་སོགས་བཞོན་པ་ལྔ། །ལག་ན་ཁར་བ་ལ་སོགས་དྲུག །ན་བ་རྣམ་པ་བཞི་རྣམས་སོ། །

# f8.c.6 ལྷུང་བཟེད་སྤྱད་པ་ལ་སོགས་པའི་སྡེ་རྣམས་ནི།
དྲུང་ན་འདུག་པའི་དགེ་སློང་གི་ལྷུང་བཟེད་ལ་འཕྱས་གདགས་པའི་ཕྱིར་མི་བལྟ་བར་བསླབ་པར་བྱ། ལག་པ་ཟས་དང་འབགས་པས་ཆུ་སྣོད་ལ་མི་བཟུང་བར་བསླབ་པར་བྱ། དྲུང་ན་འདུག་པའི་དགེ་སློང་ལ་ཟས་དང་འབགས་པའི་ཆུས་མི་གཏོར་བར་བསླབ་པར་བྱ། ཁྱིམ་པ་འདུག་པ་ལ་མ་དྲིས་པར་ཆུ་ཟས་དང་འབགས་པ་ཁྱིམ་གཞན་དུ་མི་དབོ་བར་བསླབ་པར་བྱའོ། །ལྷུང་བཟེད་ཀྱི་ནང་དུ་ཟས་ཀྱི་ལྷག་མ་བླུགས་ཏེ་མི་དོར་བར་བསླབ་པར་བྱའོ། །འོག་གཞི་མེད་པའི་ས་ཕྱོགས་སུ་ལྷུང་བཟེད་མི་བཞག་པར་བསླབ་པར་བྱ། གད་ཀ་མ་ཡིན་པ་དང་གཡང་ས་མ་ཡིན་པ་དང། དཀན་གཟར་པོ་མ་ཡིན་པར་ལྷུང་བཟེད་བཞག་པར་བསླབ་པར་བྱ། འགྲེང་སྟེ་ལྷུང་བཟེད་མི་བཀྲུ་བར་བསླབ་པར་བྱ། གད་ཀ་མ་ཡིན་པ་དང་གཡང་ས་མ་ཡིན་པ་དང། དཀན་གཟར་པོ་མ་ཡིན་པར་བཀྲུ་བར་བསླབ་པར་བྱ། འབབ་ཆུ་དྲག་པོའི་རྒྱུན་ལས་བཟློག་སྟེ་ལྷུང་བཟེད་ཀྱིས་ཆུ་མི་བཅུ་བར་བསླབ་པར་བྱའོ། །
# f8.c.7 ཆོས་བཤད་པའི་སྡེ་ནི།
མི་ན་བར་འདུག་པ་ལ་འགྲེང་སྟེ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱ། མི་ན་བར་ཉལ་བ་ལ་འདུག་སྟེ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱ། མི་ན་བར་སྟན་མཐོན་པོ་ལ་འདུག་པ་ལ་སྟན་དམའ་བ་ལ་འདུག་སྟེ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱ། མི་ན་བར་མདུན་དུ་འགྲོ་བ་ལ་ཕྱི་ནས་འགྲོ་ཞིང་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱ། མི་ན་བར་ལམ་ནས་འགྲོ་བ་ལམ་གྱི་འགྲམ་ནས་འགྲོ་ཞིང་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱའོ། །མི་ན་བར་མགོ་གཡོགས་པ་དང། བརྫེས་པ་དང་གཟར་བ་དང་། གཉའ་གོང་དུ་བསྣོལ་བ་དང་། ལྟག་པར་བསྣོལ་བ་ལ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱའོ། །མི་ན་བར་སྐྲ་དོ་ཀེར་ཅན་དང་ཞྭ་གྱོན་པ་དང་མགོ་ཅོད་པན་ཅན་དང། མགོ་ཕྲེང་བ་ཅན་དང། མགོ་དཀྲིས་པ་ལ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱའོ། །མི་ན་བར་གླང་པོ་ཆེ་ཞོན་པ་དང་རྟ་ཞོན་པ་དང། ཁྱོགས་ན་འདུག་པ་དང་ཞོན་པའི་སྟེང་ན་འདུག་པ་དང། མཆིལ་ལྷམ་གྱོན་པ་ལ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱའོ། །མི་ན་བར་ལག་ན་འཁར་བ་ཐོགས་པ་དང། ལག་ན་གདུགས་ཐོགས་པ་དང། ལག་ན་མཚོན་ཐོགས་པ་དང། ལག་ན་རལ་གྲི་ཐོགས་པ་དང། ལག་ན་དགྲ་སྟ་ཐོགས་པ་དང། གོ་ཆ་གྱོན་པ་ལ་ཆོས་མི་བཤད་པར་བསླབ་པར་བྱའོ། །
# f8.c.8 སྒྲུབ་པའི་སྡེ་ནི།
མི་ན་བར་འགྲེང་སྟེ་བཤང་གཅི་མི་བྱ་བར་བསླབ་པར་བྱ། མི་ན་བར་ཆུའི་ནང་དུ་བཤང་གཅི་དང་མཆིལ་མ་དང། སྣབས་དང། སྐྱུགས་པ་དང། རླུགས་པ་མི་དོར་བར་བསླབ་པར་བྱ། མི་ན་བར་རྩྭ་སྔོན་པོ་ཡོད་པའི་ས་ཕྱོགས་སུ་བཤང་གཅི་དང། མཆིལ་མ་དང་སྣབས་དང་སྐྱུགས་པ་དང། རླུགས་པ་མི་དོར་བར་བསླབ་པར་བྱ། 
# f8.c.9 རྒྱུ་བ་དགག་པའི་སྡེ་ནི།

# f8.e རྣམ་གཞག་བསྒྲགས་པ་ནི།

# G.2.1 གཉིས་པ་རྩོད་པ་ཞི་བྱེད།
# G.2.1.a མདོར་སྒྲོ་སྐུར་སྤང་བའི་དོན་དུ། སྡོམ་ནི།
སྡོམ་ལ། མངོན་སུམ་དྲན་པ་མ་མྱོས་དང། །དེ་བཞིན་གང་མང་ངོ་བོ་ཉིད། །རྩྭ་རྣམས་བཀྲམ་པ་ལྟ་བུ་དང། །ཁས་བླང་བར་ཡང་བྱ་བའོ། །
# G.2.1.b ལུང་འབོགས་པ་ནི།
ཚེ་དང་ལྡན་པ་དག་རྩོད་པ་ཞི་བར་བྱ་བའི་ཆོས་བདུན་པོ་འདི་དག་ནི་ཟླ་བ་ཕྱེད་ཕྱེད་ཅིང་སོ་སོར་ཐར་པའི་མདོ་འདོན་པ་ལས་འབྱུང་ངོ། །
# G.2.1.c དངོས་ཀྱི་ཚིག་དོན་ནི།
མངོན་སུམ་གྱིས་འདུལ་བར་འོས་པ་ལ་མངོན་སུམ་གྱིས་འདུལ་བ་སྦྱིན་ནོ། །དྲན་པས་འདུལ་བར་འོས་པ་ལ་དྲན་པས་འདུལ་བ་སྦྱིན་ནོ། །མ་མྱོས་པས་འདུལ་བར་འོས་པ་ལ་མ་མྱོས་པས་འདུལ་བ་སྦྱིན་ནོ། །གང་ཚུལ་ཤིང་མང་པོ་སྦྱིན་པར་འོས་པ་ལ་ཚུལ་ཤིང་མང་པོ་སྦྱིན་ནོ། །དེའི་ངོ་བོ་ཉིད་ཚོལ་དུ་གཞུག་པར་འོས་པ་ལ་དེའི་ངོ་བོ་ཉིད་བཙལ་བ་སྦྱིན་ནོ། །རྩྭ་བཀྲམ་པ་ལྟ་བུ་འོས་པ་ལ་རྩྭ་བཀྲམ་པ་ལྟ་བུ་སྦྱིན་ནོ། །གང་ཁས་བླང་བར་འོས་པ་ལ་ཁས་བླང་བར་བྱའོ། །རྩོད་པ་བྱུང་བར་གྱུར་པ་རྣམས་རྩོད་པ་ཞི་བར་བྱ་བའི་ཆོས་བདུན་པོ་འདི་དག་གིས་ཆོས་དང། འདུལ་བ་དང་སྟོན་པའི་བསྟན་པ་བཞིན་དུ་དུལ་བར་བྱ་ཞི་བར་བྱ། རྣམ་པར་ཞི་བར་བྱའོ། །
# G.2.1.d འདོན་པ་མཇུག་བསྡུ་བ་ནི།
ཚེ་དང་ལྡན་པ་དག་བདག་གིས་རྩོད་པ་ཞི་བར་བྱ་བའི་ཆོས་བདུན་པོ་དག་བཏོན་ཟིན་ཏོ། །
